### 离散模型
#### 土壤的物理性质
+ 考虑面积为$ S $，深度为$ h $的柱状土壤，土壤在水平方向上绝热，底面也绝热。设土壤密度为$ \rho $。
    - 土壤比热容设为$ C $，土壤中热导率为$ k $（单位面积上热通量与温度梯度的比值的相反数）。

$ P_z = -k\frac{\partial T}{\partial z} $

        * 其中$ T $是温度分布，$ P $是单位面积上传热功率。负号表示能量从高温传递到低温。
        * 使用离散的语言描述，就是：

$ P_{1\rightarrow 2} = - k \frac{T_2 - T_1}{|h_2-h_1|} $

#### 分层与能量交换
+ 假设单位面积上太阳辐射的变化规律为$ I_{in}(t;...) $。
+ 从上往下数，土壤总共分为$ n $层，则每层厚度为$ \Delta h = h/n $。
+ 对于最上方一块的温度：

$ C\Delta m[T_1(t+ \Delta t) - T_1(t)] = [S\alpha I_{in}(t) - Sk\frac{T_1(t)-T_2(t)}{\Delta h}  -S \sigma T_1^4(t)] \Delta t $

    - 等号右侧第一项代表太阳辐射输入，第二项代表向下传递热量输出，第三项代表黑体辐射输出。
+ 对于中间的第$ i $块的温度：

$ C\Delta m[T_i(t+ \Delta t) - T_i(t)] =  [- Sk\frac{T_i(t)-T_{i+1}(t)}{\Delta h} - Sk\frac{T_i(t)-T_{i-1}(t)}{\Delta h}] \Delta t $

    - 等号右侧第一项代表向下热量输出，第二项代表向上热量输出。
    - $ T_i(t)-T_{i+1}(t) $符取值号改变时，热量传输方向也改变，因此表达式也能处理热量输入的情形。
+ 对于最下面的一块的温度：

$ C\Delta m[T_n(t+ \Delta t) - T_n(t)] =  [- Sk\frac{T_n(t)-T_{n-1}(t)}{\Delta h}] \Delta t $

    - 等号右侧代表向上热量输出，根据底部绝热条件，不再向下输出热量。
+ 根据土壤密度$ \rho $与$ \Delta m,\Delta h,S $的关系：

$ \Delta m = \rho S \Delta h $

    - 可以将上述公式改写：

$ C\rho \Delta h [T_1(t+ \Delta t) - T_1(t)] = [\alpha I_{in}(t) - k\frac{T_1(t)-T_2(t)}{\Delta h}  - \sigma T_1^4(t)] \Delta t $

$ C\rho \Delta h[T_i(t+ \Delta t) - T_i(t)] =  [- k\frac{T_i(t)-T_{i+1}(t)}{\Delta h} - k\frac{T_i(t)-T_{i-1}(t)}{\Delta h}] \Delta t,\; i=2,3,4,5...n-1  $

$ C\rho \Delta h[T_n(t+ \Delta t) - T_n(t)] =  [- k\frac{T_n(t)-T_{n-1}(t)}{\Delta h}] \Delta t $

#### 太阳辐射规律
+ 太阳天顶角随着时间变化的规律为：

$ z(t) = \pi - \arccos (\cos \omega_2t(\sin\theta \sin \alpha +\cos\theta \cos \alpha \cos\omega_1t)+\sin\omega_2 t \cos\alpha \sin\omega_1 t) $

    - <font style="color:rgb(0, 0, 0);">当</font>$ z<\frac{\pi}{2} $<font style="color:rgb(0, 0, 0);">白天时：</font>

$ I_{in}(t;...) = I_0 \cos(h(t)) $

    - 当$ z>\frac{\pi}{2} $黑夜时：

$ I_{in}(t;...) = 0 $

        * 其中$ I_0 $为太阳常数，参考值$ 1380\mathrm{W/m^2} $。
+ <font style="color:rgb(0, 0, 0);">对于北京来说：</font>
    - <font style="color:rgb(0, 0, 0);">自转速度：</font>$ ω_1=7.292×10^{-5}\mathrm{rad/s} $
    - <font style="color:rgb(0, 0, 0);">公转速度：</font>$ ω_2=1.991×10^{-7}\mathrm{rad/s} $
    - <font style="color:rgb(0, 0, 0);">黄赤交角（弧度制）：</font>$ θ=0.405 $
    - <font style="color:rgb(0, 0, 0);">纬度（弧度制）：</font>$ α=0.698 $

```python
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
from typing import Callable
import math
from tqdm import tqdm


def solar_zenith_angle_model(
        t: float,  # time (s)
        omega1: float = 7.292e-5,  # Earth rotation speed (rad/s)
        omega2: float = 1.991e-7,  # Earth revolution speed (rad/s)
        theta: float = 0.405,  # Obliquity of the ecliptic (rad)
        alpha: float = 0.698  # Latitude (rad) - Beijing
) -> float:
    """
    Calculate solar zenith angle

    Parameters:
    - t: time (s)
    - omega1: rotation speed (rad/s)
    - omega2: revolution speed (rad/s)
    - theta: obliquity (rad)
    - alpha: latitude (rad)

    Returns:
    - Solar zenith angle (rad)
    """
    # Calculate solar zenith angle h(t)
    term1 = np.cos(omega2 * t) * (np.sin(theta) * np.sin(alpha) +
                                  np.cos(theta) * np.cos(alpha) * np.cos(omega1 * t))
    term2 = np.sin(omega2 * t) * np.cos(alpha) * np.sin(omega1 * t)

    cos_h = term1 + term2

    # Ensure cos_h is in valid range [-1, 1]
    cos_h = np.clip(cos_h, -1, 1)

    h = np.pi - np.arccos(cos_h)
    return h


def solar_radiation_model(
        t: float,  # time (s)
        I0: float = 1380.0,  # Solar constant (W/m²)
        omega1: float = 7.292e-5,  # Earth rotation speed (rad/s)
        omega2: float = 1.991e-7,  # Earth revolution speed (rad/s)
        theta: float = 0.405,  # Obliquity of the ecliptic (rad)
        alpha: float = 0.698  # Latitude (rad) - Beijing
) -> float:
    """
    Calculate solar radiation intensity

    Parameters:
    - t: time (s)
    - I0: solar constant (W/m²)
    - omega1: rotation speed (rad/s)
    - omega2: revolution speed (rad/s)
    - theta: obliquity (rad)
    - alpha: latitude (rad)

    Returns:
    - Solar radiation intensity (W/m²)
    """
    # Calculate solar zenith angle
    h = solar_zenith_angle_model(t, omega1, omega2, theta, alpha)

    # Calculate radiation intensity based on zenith angle
    if h < np.pi / 2:  # Daytime
        return I0 * np.cos(h)
    else:  # Nighttime
        return 0.0


def soil_temperature_simulation(
        # Physical constants
        rho: float = 1600.0,  # Soil density (kg/m³)
        k: float = 0.5,  # Soil thermal conductivity (W/(m·K))
        sigma: float = 5.67e-8,  # Stefan-Boltzmann constant (W/(m²·K⁴))
        alpha: float = 0.7,  # Surface absorptivity (dimensionless)
        C: float = 800.0,  # Soil specific heat capacity (J/(kg·K))

        # Geometric parameters
        H: float = 2.0,  # Soil column total depth (m) - increased for annual simulation
        S: float = 1.0,  # Soil column cross-sectional area (m²)

        # Spatial and temporal resolution
        dz: float = 0.05,  # Spatial resolution (m) - increased for efficiency
        total_time: float = 365 * 24 * 3600,  # Total simulation time (s) - 1 year
        dt: float = 1800.0,  # Time step (s) - 30 minutes for better stability

        # Solar radiation parameters
        I0: float = 1380.0,  # Solar constant (W/m²)
        omega1: float = 7.292e-5,  # Earth rotation speed (rad/s)
        omega2: float = 1.991e-7,  # Earth revolution speed (rad/s)
        theta: float = 0.405,  # Obliquity of the ecliptic (rad)
        latitude: float = 0.698,  # Latitude (rad) - Beijing

        # Initial condition function
        initial_temperature: Callable[[float], float] = lambda z: 15.0,  # Initial temperature distribution (°C)

        # Output frequency
        output_frequency: int = 48,  # Output every 48 steps (daily)
):
    """
    Soil temperature discrete layer model simulation for one year

    Parameters:
    - rho: Soil density (kg/m³)
    - k: Soil thermal conductivity (W/(m·K))
    - sigma: Stefan-Boltzmann constant (W/(m²·K⁴))
    - alpha: Surface absorptivity
    - C: Soil specific heat capacity (J/(kg·K))
    - H: Soil column total depth (m)
    - S: Soil column cross-sectional area (m²)
    - dz: Spatial resolution (m)
    - total_time: Total simulation time (s)
    - dt: Time step (s)
    - I0: Solar constant (W/m²)
    - omega1: Earth rotation speed (rad/s)
    - omega2: Earth revolution speed (rad/s)
    - theta: Obliquity of the ecliptic (rad)
    - latitude: Latitude (rad)
    - initial_temperature: Initial temperature distribution function f(z) -> T (°C)
    - output_frequency: Output data every N time steps
    """

    # 1. Calculate number of layers and time steps
    n_layers = int(H / dz)  # Number of soil layers
    n_time_steps = int(total_time / dt)  # Number of time steps
    n_output_steps = n_time_steps // output_frequency + 1  # Number of output steps

    print(f"Number of soil layers: {n_layers}")
    print(f"Number of time steps: {n_time_steps}")
    print(f"Number of output steps: {n_output_steps}")
    print(f"Layer thickness: {dz} m")
    print(f"Time step: {dt} s ({dt / 3600} hours)")
    print(f"Output frequency: every {output_frequency} steps ({output_frequency * dt / 3600} hours)")

    # 2. Initialize temperature arrays
    # For efficiency, we only store output steps
    temperature_output = np.zeros((n_output_steps, n_layers))
    current_temperature = np.zeros(n_layers)

    # Set initial conditions
    for i in range(n_layers):
        depth = (i + 0.5) * dz  # Center depth of each layer
        current_temperature[i] = initial_temperature(depth)

    temperature_output[0, :] = current_temperature

    # 3. Iterative calculation with progress bar
    output_idx = 1

    # Create progress bar
    pbar = tqdm(total=n_time_steps, desc="Simulation Progress")

    for step in range(n_time_steps):
        current_time = step * dt
        I_m = solar_radiation_model(current_time, I0, omega1, omega2, theta, latitude)

        # Create a copy for the next time step
        next_temperature = np.copy(current_temperature)

        # Calculate temperature for next time step
        for i in range(n_layers):
            current_temp = current_temperature[i]

            if i == 0:  # Top layer
                # Solar radiation + downward heat conduction + blackbody radiation
                T_below = current_temperature[1]

                # Blackbody radiation uses Kelvin temperature (°C + 273.15)
                T_kelvin = current_temp + 273.15

                # Check for valid temperature to avoid overflow
                if T_kelvin > 0:
                    blackbody_radiation = sigma * T_kelvin ** 4
                else:
                    blackbody_radiation = 0

                heat_flux = (alpha * I_m -
                             k * (current_temp - T_below) / dz -
                             blackbody_radiation)

            elif i == n_layers - 1:  # Bottom layer (adiabatic boundary)
                # Only upward heat conduction
                T_above = current_temperature[i - 1]
                heat_flux = -k * (current_temp - T_above) / dz

            else:  # Middle layers
                # Upward and downward heat conduction
                T_above = current_temperature[i - 1]
                T_below = current_temperature[i + 1]
                heat_flux = (-k * (current_temp - T_below) / dz -
                             k * (current_temp - T_above) / dz)

            # Update temperature (explicit Euler method)
            # Add stability check to prevent overflow
            temp_change = (dt / (C * rho * dz)) * heat_flux
            if np.isfinite(temp_change) and abs(temp_change) < 1000:  # Reasonable temperature change limit
                next_temperature[i] = current_temp + temp_change
            else:
                next_temperature[i] = current_temp  # No change if unstable

        # Update current temperature
        current_temperature = next_temperature

        # Store output if needed
        if (step + 1) % output_frequency == 0 and output_idx < n_output_steps:
            temperature_output[output_idx, :] = current_temperature
            output_idx += 1

        # Update progress bar
        pbar.update(1)

    pbar.close()

    # 4. Create output DataFrame
    time_array = np.arange(0, n_output_steps) * output_frequency * dt
    depth_array = np.array([(i + 0.5) * dz for i in range(n_layers)])

    # Create column names
    column_names = ['Time (s)', 'Time (days)']
    for depth in depth_array:
        column_names.append(f'Depth {depth:.2f}m (°C)')

    # Create DataFrame
    df = pd.DataFrame(temperature_output, columns=[f'Depth {depth:.2f}m (°C)' for depth in depth_array])
    df.insert(0, 'Time (days)', time_array / (24 * 3600))
    df.insert(0, 'Time (s)', time_array)

    return df, temperature_output, depth_array, time_array, n_layers, n_output_steps


def save_to_excel(df, filename="soil_temperature_annual_simulation.xlsx"):
    """Save results to Excel file"""
    try:
        # Ensure DataFrame is not empty
        if df.empty:
            print("Warning: DataFrame is empty, cannot save to Excel")
            return

        # Check if there is valid data
        if df.iloc[:, 2:].isna().all().all():
            print("Warning: All temperature data is NaN, cannot save to Excel")
            return

        df.to_excel(filename, index=False, float_format="%.3f")

        print(f"Data shape: {df.shape}")
    except Exception as e:
        print(f"Error saving Excel file: {e}")
        import traceback
        traceback.print_exc()


def plot_annual_solar_parameters():
    """Plot solar altitude angle and solar radiation over one year"""
    # Create time array (1 year, in seconds)
    days_in_year = 365
    t_seconds = np.linspace(0, days_in_year * 24 * 3600, 1000)  # 1 year, 1000 points
    t_days = t_seconds / (24 * 3600)  # Convert to days

    # Calculate solar zenith and altitude angles
    zenith_angles = [solar_zenith_angle_model(t) for t in t_seconds]
    altitude_angles = [np.pi / 2 - z for z in zenith_angles]  # Altitude angle = π/2 - zenith angle

    # Convert to degrees for readability
    altitude_degrees = np.degrees(altitude_angles)

    # Calculate solar radiation intensity
    radiation = [solar_radiation_model(t) for t in t_seconds]

    # Create figure
    plt.figure(figsize=(15, 10))

    # Plot solar altitude angle over year
    plt.subplot(2, 2, 1)
    plt.plot(t_days, altitude_degrees, 'g-', linewidth=1)
    plt.xlabel('Time (days)')
    plt.ylabel('Solar Altitude Angle at Noon (degrees)')
    plt.title('Solar Altitude Angle Variation in Beijing (1 Year)')
    plt.grid(True, alpha=0.3)

    # Plot solar radiation intensity over year
    plt.subplot(2, 2, 2)
    plt.plot(t_days, radiation, 'r-', linewidth=1)
    plt.xlabel('Time (days)')
    plt.ylabel('Solar Radiation Intensity at Noon (W/m²)')
    plt.title('Solar Radiation Intensity Variation in Beijing (1 Year)')
    plt.grid(True, alpha=0.3)

    # Plot daily max solar altitude angle
    plt.subplot(2, 2, 3)
    # Calculate noon time for each day (assuming solar noon at 12:00)
```
