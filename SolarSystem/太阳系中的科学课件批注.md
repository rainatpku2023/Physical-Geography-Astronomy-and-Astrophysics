### Lecture 2: Celestial Orbits
+ 谷神星的故事
    - 谷神星不是恒星<-相对背景星空有运动
        * 其他判断方法：恒星距离较远，为点状光源，大气扰动时容易发生闪烁；行星为面状光源，大气扰动影响有限，难以发生闪烁
    - 朱塞佩·皮亚齐（1801）：发现谷神星
    - 高斯：重现发现谷神星<-最小二乘法<-22个时间点的数据，6个待求轨道参数
    - 提丢斯·博德（1766）：错误的提丢斯·博德定则->谷神星被认为是行星
    - 1851：谷神星的降级
+ 海王星的故事
    - William Herschel（1781）：发现天王星
    - Le Verrier 和Adams：天王星轨道异常
    - Galle（1846）：笔尖上发现海王星
+ 冥王星的故事
    - Percival Lowell：海王星与预测轨道偏离->建立Lowell天文台寻找Planet X
    - Clyde Tombaugh（1930）：冥王星的发现
    - 1915-1978：冥王星质量的估算，估算值越来越小
    - 冥卫一（Charon，卡戎）的发现
    - Robert Harrington（1978-1992）：再度寻找Planet X，旅行者号发现不需要Planet X就能解释轨道
    - 2005：阋神星的发现
    - IAU行星定义（2006）：冥王星降级

### Lecture 3: Tides
+ 地球上的潮汐
    - 潮汐定义：地球上的海洋准周期性涨落现象
    - 大潮与小潮：大潮潮水最大，小潮潮水最小；每个月均为两次；大潮又称朔望朝
+ 潮汐力
    - 意大利面条化
    - 流体洛希极限->潮汐力与自身引力相比；适用于流体
    - 潮汐力相关现象：慧木大碰撞（1994.7），土星环的发现（1612），潮汐锁定
        * 土星环
            + 伽利略的疑惑（1612/1616）：为什么有时能看见有时看不见
            + 惠更斯的猜想：薄而平坦的环
            + 卡西尼的发现：卡西尼环缝
            + 拉普拉斯的猜想：木星环由小环组成
            + 麦克斯韦证明：大量小环仍然不稳定，土星环应该是由很多个小颗粒组成
        * 潮汐锁定：环绕天体始终都以同一面朝向中心天体
            + 月球的潮汐锁定：地球自转变慢、月球公转速度提高、轨道膨胀
            + 水星的潮汐锁定未发生<-3：2的公转：自转轨道共振
            + Europa潮汐锁定未发生<-三个伽利略卫星的拉普拉斯共振（Io（木卫一），Europa（木卫二），Ganymede（木卫三）)
                - Europa含水量极大
                - Europa南极附近存在间歇性喷泉
                - 潮汐作用导致围绕天体地质活动增加
            + 海卫一（Triton）逆行：过去（捕获？），现在，未来（解体、撞向海王星）

### Lecture 4: Space Era
+ 进入太空的标准：穿过卡门线（100km，天空和太空的分界线）
+ 完整的太空计划：任务目标+航天器（卫星）+晕在系统（火箭）+轨道设计+测控系统
+ 太空时代历程
    - V-2火箭（1944.6）：第一个穿过卡门线的人造物体
    - Sputnik-1（1957.10.4）：第一颗人造卫星的发射->太空时代开启的标志
        * 火箭的基本原理：齐奥尔科夫斯基火箭方程（参考普物）

$ v_1-v_0 = v_e \cdot ln\frac{m_0}{m_1} $

    - Sputnik-2：第二颗人造卫星，第一颗携带科学仪器的卫星（徕卡、盖革计数器、分光光度计）
        * 险些发现辐射带
    - Explorer I：第三颗人造卫星，美国的第一颗人造卫星
        * 范艾伦发现（内）辐射带
    - Sputnik-3：发现外辐射带->内辐射带和外辐射带之间为槽区
+ 范艾伦辐射带
    - 电离辐射
        * 辐射：能量以波或者粒子的形式在空间或介质中的发射与传播
        * 电离辐射：高能量射线可将原子电离，即将电子从中分离出来
        * 电离辐射的探测：威尔逊云室、盖革计数器
    - 辐射探测历史
        * Victor Hess：5000米高空，辐射强度显著高于地面，日全食时并无明显下降，获得诺贝尔奖（1936），宇宙射线命名
        * Arthur Compton：带电粒子通量随纬度的分布
        * Sergei Vernov和James Van Allen：宇宙学专家，安装盖革计数器
    - 地磁场对于高能粒子的空间约束
        * 高空核试验：Starfish Prime（1962）
    - 地球磁轴的不穿过球心->南大西洋/南美洲出现弱磁场区
    - 木星的辐射带->人类只能重装铠甲+轨道规避

### Lecture 14: Stellar Evolution
+ 太阳的一生：分子云 -> 原始星 -> 主序星 -> 红巨星 -> 白矮星 -> 黑矮星
+ 问题：如何得知太阳/恒星的命运？
+ 恒星分类：哈佛分类法 O-B-A-F-G-K-M
    - 按照颜色分类，从O到M越来越红 -> 温度越来越低（维恩位移定律）
+ 恒星的亮度：绝对星等与视星等
    - 织女星视星等为0
    - 视星等每增加5，天体亮度增加99%
+ 恒星的距离
    - 计算：利用两只眼睛观察物体的角度差，获得立体视觉 -> 不同季节看，相当于两只眼睛（举例2个AU）
+ 赫罗图：温度和光度作为横纵坐标
    - 主序星：赫罗图对角线附近的大量恒星
    - 红巨星或红超巨星：光度高但是温度低 -> 说明体积比较大
    - 白矮星：光度低但是温度高 -> 说明体积比较小
+ 恒星的质量 -> 体积较大的恒星不一定对应于较大的质量
    - 天狼星b：体积略小于地球，质量略大于太阳 -> 质量由a和b轨道参数计算而成
    - 主序星光度与质量正相关

$ P ∝ M^{3.5}（近似） $

+ 恒星的寿命
    - 质量越大的主序星，光度越大，表面温度越高，但预期寿命越小

$ 寿命∝\frac{核燃料速度}{核反应速度}∝\frac{M}{P}∝\frac{1}{M^{2.5}} $

    - 观察方式：一个星团中包含大量诞生时间相仿、位置接近的恒星
    - 恒星的演化路径，如果足够大：氢 -> 氦 -> 氧、碳 -> 铁 -> 铀（？）
        * 氦闪，发生在恒星正中心位置
    - 恒星的归宿：取决于其质量
