### 天体物理过程的信使（Messenger）
+ 历史
    - 早就实现的：光子、中微子、宇宙线
        * 宇宙线：氢、氦、氧原子核等等。地上和宇宙中的比例不同。目前观测到的能有300EeV的宇宙线（OMG事件）。
    - 刚刚实现的：引力波（引力子）
+ 光子（电磁波）探测是目前天体物理学家获取信息的主要手段
    - 能量从高到低：γ射线，X射线，紫外线，可见光，红外射线射电波。
    - 高能的测量方式，更偏向于射线测量（粒子测量）
    - 低能的测量方式，更偏向于波动测量
+ 辐射的类型
    - 热辐射：处于热平衡的物体所发射的辐射
        * 主要成分是热辐射的光子
    - 非热辐射：未处于热平衡物体的辐射
        * 比如磁场环境下非热高能电子辐射：回旋辐射、同步辐射、曲率辐射
            + 把磁场看成光子，这也可以认为是电子场和光子场的相互作用。
        * 其他类型的：Compton辐射、韧致辐射、Cherenkov辐射
            + Compton辐射：光子与电子相互作用交换动量能量。
            + 韧致辐射：电子在电场中加速产生的辐射
            + Cherenkov辐射：电子进入介质激发周围物质产生的辐射，介质中光速低于电子运动速度，即电磁激波

### 大气辐射窗口
+ 经过地球大气层的吸收，只有3个频段能够透射。
    - 光学窗口：~300nm 至 ~900nm，**实际上比可见的部分还要大一点**。
    - 红外窗口：由若干μm波长的窄波段构成
    - 射电窗口： ~1mm 至 ~30m（~10MHz 至 ~300GHz）
        * 波长再长一点就被电离层吸收了。电离层是宇宙线轰击地球大气层顶端形成的层，里面的带电粒子可以吸收。

### 热辐射
#### 黑体辐射
+ Kirchhoff定律：能量的发射速率和吸收速率成正比

$ \psi_e(\nu,T) = \alpha (\nu,T) \cdot B (\nu,T) $

    - 其中$ B(\nu,T) $是与材料无关的函数。
    - 灰体：如果发射系数$ \alpha $与频率无关，而且小于1，那么称为灰体。
+ 黑体辐射：吸收系数$ \alpha (\nu,T ) = 1 $的热辐射
+ 能量密度：Planck公式

$ \rho_\nu (T) = \frac{8\pi h \nu^3}{c^3}\frac{1}{e^{h\nu/k_B T}-1} $

+ 辐射通量：Stefan-Boltzmann定律

$ B(T) = \frac{c}{4}\int \rho_V (T) d\nu = \sigma T^4 $

+ Wien位移定律

$ \lambda_{max} T = 0.29 \mathrm{cm \cdot K} $

+ 辐射场的状态方程：

$ p(T) = \rho (T)/3 $

    - 宇宙的状态方程决定了宇宙如何演化。
        * 在宇宙初期高温高密辐射相$ p(T) = \rho (T)/3 $。
        * 在宇宙晚期低温低密尘埃宇宙$ p=0 $。
        * 宇宙加速膨胀，需要负压强。

### 非热辐射
#### 回旋辐射（电子非相对论性）
+ 典型非热辐射：电子加速运动产生的辐射
+ 为什么μ子探测器要埋起来：因为μ子和电子电量相同，但是μ子质量比较大，减速比较慢。
+ 圆周运动：可以分解为相位差$ \pi/2 $的两个互相垂直电偶极子，一个在$ x $轴上振荡，另外一个在$ y $轴上振荡。
    - 单色的：偶极子振荡产生的电磁波是单色的，所以叠加结果也是单色的。
    - 辐射近乎各向同性：大概差2倍左右无所谓。
    - 椭圆偏振：因为有$ \vec{S}\sim \sin^2 \theta \vec{n} $，在垂直振荡方向发射最多，$ x,y $方向上是线偏振的，根据对称性$ Oxy $平面上都是线偏振。$ z $方向上是圆偏振的。其他的位置介于二者之间是椭圆偏振的。
    - γ不太大的也归入回旋辐射中，要考虑狭义相对论效应

+ 最大动能和极端事件的空间尺度相关，如果动能太大就跑出去了不能继续加速，所以实际上最大动能是在极端事件的空间边界上产生的。经过计算可知，最大能量大概是$ qBR $量级的。
+ 回旋辐射的功率较小，正比于$ \beta^2B^2 $。

#### 同步辐射（电子极端相对论性）
+ 相对论性电子产生的辐射，功率较大，正比于$ \gamma^2\beta^2B^2 $
    - 根据狭义相对论效应，大部分辐射功率都分布在运动方向的一个小角度内。

<img src="https://cdn.nlark.com/yuque/0/2026/png/2371504/1773299770170-8c30a016-6ef5-4fe7-9eb6-9b0e1b000e8a.png" width="399" title="" crop="0,0,1,1" id="ud3b87c0d" class="ne-image">

+ 时域信号：根据狭义相对论，能量只在一个小锥内，然后电子转过去之后就接受不到信号，过了一个周期后又能看见，所以周期可以用于估测gamma。
    - 傅立叶变换的一个二级结论：频域中峰值频率大概是最强脉冲成分时域半高宽的倒数。
    - 根据时域半高宽又可以估测gamma。
        * 为什么是gamma三次方：一次方是因为锥的角度，二次方是因为电子相对于光速不可忽略。

#### 曲率辐射（强磁场内的运动电子）
+ 磁场强中相对论电子运动的量子效应
+ 有一个定理：非奇异势能场内束缚态的能量是离散谱。氢原子虽然是奇异势能场，但是也是离散谱。（参考曾谨言《量子力学教程》）
    - 相对论性电子被洛伦兹力约束磁场中，也会有分立的能级，即Landau能级。
    - 如果约束范围小于电子波动波长，则存在明显能级

#### Compton散射与逆Compton散射（电子与光子相互作用）
+ Compton过程：自由电子与光子之间的碰撞过程。
    - Compton 散射：电子动能<<光子能量，光子将能量交给电子
    - 逆Compton散射：电子动能>>光子能量，电子将能量交给光子产生高能光子
        * 逆Compton散射别称为：Compton辐射
+ Thomson散射：能量<511keV光子被几乎静止电子散射
    - 光子表现波动性而电子显示粒子性，这个时候就变成了经典电动力学的问题。
    - 可以近似为电偶极辐射，角分布为$ \sin^2\theta $。
    - 散射后光子的动量大小不变，方向改变，因为角分布为$ \sin^2\theta $，最后总动量为0。
    - 所以最后相当于原来定向光子流的全部动量都给了电子。
    - 电子的散射截面是$ \sigma_T $。

<img src="https://cdn.nlark.com/yuque/0/2026/png/2371504/1773304300635-268de9e3-98d8-4576-9bf0-c8c4952927e6.png" width="400" title="" crop="0,0,1,1" id="u9dfca3cc" class="ne-image">

+ Eddington光度：对称引力束缚体最大光度/恒星亮度上限
    - 恒星内部物质是电离的状态，电子受到光子辐射会往外跑。
    - 恒星存在，电子不能跑，因此电子受到的辐射不能太大。
    - 辐射压强与引力相平衡，得到的光度就是Eddington光度。
+ Eddington吸积率：球对称引力束缚体最快吸积的速度
    - 假设吸积物质各向同性地匀速坠落在球形天体表面，引力势能以黑体辐射形式释放。
    - 这个黑体辐射不能太强，否则电子就跑了。
    - 这个黑体辐射和引力相平衡，得到辐射强度，再转化为吸积率，得到的吸积率就是Eddington吸积率。
    - **跑出去的电子已经考虑在**$ \dot{M} $**中了，电子抛出去需要辐射能转化为动能转化为引力势能，相当于逆吸积过程，所以可以直接画等号。**

+ 逆Compton散射的结果：出射光子频率$ \nu' \sim \gamma^2\nu $，几乎沿电子方向。
+ **用于探测的高能光子有上限，因为如果能量太高，和3K背景辐射相互作用，会打出正负电子对。**
    - 虽然地面系看3K背景辐射的能量很低。
    - 但是可以取高能光子和3K背景辐射中点的系，这个时候两个光子的能量都很高，会产生正负电子对。
    - 产生正负电子对的事件与坐标系无关，所以也会观察到正负电子对。
