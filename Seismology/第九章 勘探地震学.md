+ 石油勘探的三种方法：地球物理方法/地质法/钻探法
    - <font style="color:#8A8F8D;">三个环节：野外采集阶段、室内资料处理阶段、地震资料的解释阶段</font>
+ 地质法：俗称地质勘探
    - 看到有形成的同时代的岩石就可以推断可能存在石油
    - 地质勘探获取的资料来自于地标，虽然对浅层的地质情况能够有比较详尽的了解，但是对于深层的地下结构最多只能有个大概的限定
+ 地球物理方法：简称物探
    - **<font style="color:#DF2A3F;background-color:#FBDE28;">几种方法：重力勘探/磁性勘探/电法勘探/地震勘探</font>**
+ 钻探法
    - 和地震方法相比，它能够直接获得深层地下的物质情况，所以可信度更高，然而由于钻探成本的昂贵以及覆盖范围的限制，在实际生产中更多的是作为辅助性的资料
+ 勘探地震学
    - 天然地震学：被动源方法，观测天然地震产生的地震波场，范围较广
    - 勘探地震学：主动源方法，利用人工产生的地震波场，范围较窄
    - 反射地震学/折射地震学：折射情况较为复杂，需要已知内部结构，计算方法较为繁琐，最终被反射地震法取代
        * **<font style="color:#DF2A3F;background-color:#FBDE28;">先为折射法，后被反射法取代</font>**
+ 地震勘探技术三个环节
    - 野外采集阶段
    - 室内资料处理阶段
    - 地震资料的解释阶段（核心工作）
+ 地震速度的概念：层速度/平均速度/均方根速度/叠加速度
    - 层速度：最基本的概念
    - 平均速度：总传播路程除以总的传播时间
    - 均方根速度：Dix公式
    - 叠加速度：将复杂的地下速度模型等效成一个单层的速度模型，会在叠加和偏移阶段使用
    - 检波器：本质上就是地震仪
    - 速度的测量：地震测井法、声波测井方法、VSP方法
+ 地震资料的处理
    - 野外地震资料的编辑：观测系统的加载、坏道和初至的剔除、增益的恢复、静校正
    - 去噪处理：去除一次反射波以外的杂波
    - 速度分析
    - 叠加偏移
+ 地震资料的解释：最核心的部分
    - **<font style="color:#DF2A3F;background-color:#FBDE28;">地震资料解释精度只有15-25%</font>**
    - 地震解释是一个多解问题，即我们无法保证解释出来的成果一定是正确的
    - 时频分析技术/多波多分量技术/正演技术和反演技术
+ 勘探地震学与天然地震学的不同点：研究对象不同，地震采集的方式不同，地震采集的仪器也有所差别
