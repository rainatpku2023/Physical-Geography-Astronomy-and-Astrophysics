+ 地震的危害：全世界有6亿多人生活在强地震带上，20世纪越有200万人思域地震，随着人口密度的加大，预计21世纪将有约1500万人死于地震
+ 我国地震历史：20世纪以来我国发生了800多次6级以上的地震，平均每年约8次；全球死亡超过20万人的地震有6次，其中在中国就有4次
+ 地震可能伴生着许多次生灾害，如火灾等
+ 什么是地震学：地震学是关于地震的科学，它是以地震资料为基础，用数学、物理和地质知识研究地震机理即地震波传播的规律，以防御地震灾害、研究地壳和地球内部构造以及促使研究结果在经济建设和国防建设中得以应用
+ 地球科学-地球物理学-固体地球物理学-地震学
+ 地震学是一门应用物理学
+ 应用：地下核爆破的检测
+ 地震学研究方面：
    - 宏观地震学
    - 微观地震学：地震波的传播理论，测震学
+ 地震的基本概念：地球内部介质突然发生破坏，产生地震波，并在相当范围内引起地面震动的现象
+ 相关名词：震源、震中、震源深度、震中距、烈度、震级、地震序列、发震时刻（格林尼治时间）
    - 地震序列分类：主震型、震群型
    - 天然地震分类
        * 按照成因分：构造地震、火山地震、陷落地震
        * 按照震源深度分：浅、中、深源地震；60km和300km分界线；有时也将中源地震和深源地震统称为深震

        * 按照震级分类：弱震、有感地震、中强震、强震；3级、4.5级、6级、8级
