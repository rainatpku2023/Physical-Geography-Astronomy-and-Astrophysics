### 第一节 烈度和地震烈度区划
+ 震级：反映一次地震本身大小的等级，用M表示
    - 表达式：A表示标准地震仪距震中100km记录的最大水平地动位移，单位为微米（10^-6米）

$ M = \log{A} $

    - 震级与能量的关系
        * 表达式：能量E的单位为尔格，1尔格等于10^-7J

$ \log{E} = 11.8 + 1.5 M $

        * 关系：能量越大，震级就越大；震级相差一级，能量相差约32（31.6）倍；相差两级，能量相差1000倍
        * 对应关系：微震2级以下，有感地震2-4级，破坏性地震5级以上，强烈地震7级以上，特大地震8级以上
+ 烈度（Seismic Intensity）：一次地震对某一地区的影响和破坏程度称为地震烈度，简称烈度，用I表示
    - 影响因素：震级、震中距、震源深度、地质构造、地基条件
        * 震级描述的是一次地震整体的能量释放情况
        * 烈度描述的事一次地震对某个点的破坏程度
+ 极震区：一般震中所在的地区烈度最高，称为极震区
+ 烈度表：把人对地震的感觉、地面及地面上房屋器具、工程建筑等遭受地震影响和自然破坏的各种现象，按照不同程度划分等级，依次排列成表
    - 1964年MSK烈度表：梅德韦杰夫（S.V. Medvedev），施蓬霍伊尔（W.Sponheuer），卡尔尼克（V. Karnik）
    - 1980年《中国地震烈度表（1980）》引入加速度和速度的尺度
+ 地震烈度区划
    - 烈度定量问题
    - 基本烈度：一个地区未来50年内一般场地下可能遭受的具有10%超越概率地震烈度值
        * 相当于475年一遇的最大地震的烈度
        * 也称为偶遇烈度或中震烈度
    - 1976年《中国地震烈度区划图》，比例尺1：3 000 000，用等烈度线表示出从1974年至2073年100年内各地区可能普遍遭遇的最大地震烈度
        * 原则和方法：以地震区和带作为区划的基本单元，运用综合分析确定百年地震趋势和地震危险区，结合地震宏观影响场的分析，进行地震烈度区划

### 第二节 地震的宏观现象和宏观调查
+ 宏观现象：人类可以感觉活着观察到的各种自然现象，又分为原生现象和次生现象两种
    - 原生现象：与地震成因有直接关系
        * 如大断层的发生和重新活动、大块地面的倾斜、升降及变形等
    - 次生现象
        * 如地面的滑坡和裂隙、建筑物的破坏、人和动物的反应等
+ 微观现象：人类必须借助仪器才能观测到的各种物理场的变化
    - 地球介质的震动、地磁场的异常、重力场的变化、地光、地电、地下水位的升降和氡气含量的变化
+ 地震宏观调查：在地震现场对人能直接感觉到的地震现象包括地震所造成的破坏和影响所进行的实地调查
    - 主要内容
        * 强地震的调查：调查极震区内发震构造（Seismogenic Structure）的活动情况
        * 前兆现象的调查：小地震情况，井泉水位的变化，气候变化状况，大小动物的异常情况 -> 为之后的地震预报做准备
        * 历史地震的调查与考证：考察古建筑物受到的破坏情况
        * 宏观地震资料的整理：绘制等震线图，确定宏观震中（极震区的几何中心）

### 第三节 决定强地面震动的因素
+ 地面运动的一般特征的衡量
    - 地面最大的加速度
    - 地面运动的周期
    - 强震的持续时间
+ 影响地面震动强度的因素
    - 震源机制
    - 震点与该场点之间岩石的不均匀性和结构变化
    - 该场点的土壤和其他地质条件
+ 特殊地质环境的影响
    - 走滑断层的面特别弯曲不平处，断层将产生突然增大的高频能量脉冲
    - 不均匀的地壳岩石和山脉、溪谷等陡峭地形会使波长几百米的高频地震波，被散射或放大
    - 厚的冲积土壤可能放大某些波而削弱另一些波，这取决于土壤、岩石结构以及波的频率
    - 基岩上地震动幅值小、持续短、震害轻
    - 淤泥和填充地则放大地震波
    - 盆地效应和盆地边缘效应
+ 一些例子
    - 1989年美国罗马普瑞特7.1级地震：基岩和人工填充地上检测台镇记录到的地震图
    - 1985年墨西哥8.1级地震：黏土层固有周期2秒，放大振幅；10-14层建筑固有周期2秒，放大振幅；37层楼固有周期3.7秒，不会共振
+ 地震波：地震发生时由震源地方的岩石破裂产生的弹性波
    - 横波：周期长，振幅大，波速慢，3-6km/s
    - 纵波：周期短，振幅小，博速快，6-10km/s
    - 面波：衰减慢，振幅大，周期长，传播远，是建筑物破坏的主要因素
+ 地震的破坏作用（震害现象）
    - 直接灾害：由地震的原生灾害如底层错动，以及地震比引起的强烈地面震动所造成的灾害
        * 包括地面破坏，建筑物与构筑物的破坏，山体等自然物的破坏，海啸
    - 次生灾害：直接灾害发生后，破坏了自然或社会原有的平衡、稳定状态，从而引发出的灾害
        * 包括火灾，水灾，毒气泄漏，瘟疫
        * 有时次生灾害所造成的伤害和损失比直接灾害还大
    - 工程结构的破坏现象：结构丧失整体性，承重结构强度不足，结构性变过大导致倒塌，结构构件链接支撑失效，地基失效，非结构构件破坏

### 第四节 工程地震学
+ 减轻地震灾害的两种途径：地震预报、地震工程途径
+ 工程地震学（Engineering Seismology）：地震学中为工程建设服务的一个分支，主要研究强烈地震运动及其效应
    - 1931日本地震学者末广恭二赴美国讲学，讲题为“工程地震学”
    - 1963年苏联梅德韦杰夫出版《工程地震学》，内容包括地震区划和小区划以及结构在地震作用下的反应
    - 1983年日本又出版了金井清著的《工程地震学》，涉及了地震观测、地震活动性、地面和建筑物的振动、地震破坏现象、工程抗震设计准则等方面。
+ 工程地震学研究主题：地震宏观考察，强震观测，近场地面运动，地震区域划分和地震危险性分析，地震小区划，近场地震学
