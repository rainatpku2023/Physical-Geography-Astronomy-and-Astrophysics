### 第一节 断层
+ 断层：沿破裂面两侧岩块发生显著相对位移的断裂构造
    - 断层面：岩块沿之发生相对位移的破裂面
+ 尺度特征：大小长度差异很多，宽度大的可能成为破碎带
+ 产生原因：地壳运动中产生的强大的应力（压力和张力）超过岩石承受极限使岩石破裂
    - 一些几何术语：断层上盘/断层下盘/走向φs/倾角δ/滑移/位移（真位移与视位移）
        * 走向：正北顺时针转，上盘在右侧（例如正东为90度）
        * 倾角：断层面与水平面的夹角
        * 滑移方向：断层的上盘相对于下盘滑动的方向，这时断层类型的划分标准
            + 正断层：上盘相对于下盘向下滑动（Normal）
            + 逆断层：上盘相对于下盘向上滑动（Reverse）
                - 冲断层：断层面倾角小于30度的逆断层
            + 走滑断层：水平移动（Strike-Slip）
                - 左旋走滑断层/右旋走滑断层：在俯视角度，看两盘是左旋就左旋走滑断层（或者站在一盘看另外一盘是向左就是左旋走滑断层）
            + 斜滑断层：既有水平位移又有垂直位移的（Oblique）
    - <font style="color:#8A8F8D;">野外认识断层及其性质的主要标志</font>
        * <font style="color:#8A8F8D;">地层、岩脉、矿脉等地质体在平面或剖面上突然中断或错开</font>
        * <font style="color:#8A8F8D;">低层的重复或缺失</font>
        * <font style="color:#8A8F8D;">擦痕，断层面上两盘岩石相互摩擦留下的痕迹</font>
        * <font style="color:#8A8F8D;">牵引构造，岩层受到拖拽造成的局部弧形弯曲，凸出方向大体指示了所在盘的相对运动方向</font>
        * <font style="color:#8A8F8D;">断层两盘岩石碎块构成的断层角砾岩、断层运动研磨成粉末状断层泥等的表现表面该处存在断层</font>

### 第二节 弹性回跳原理
+ 岩石变形：冷脆的时候容易发生脆性破裂，从而导致天然地震；温度较高的受力情况下容易发生弯曲和流动 -> 地球深部没有地震的原因
    - 如果应力撤销时，岩石又能回复到原先的状态，这时岩石的表现叫弹性体
    - <font style="color:#8A8F8D;">本质上是分子的定居时间和外界作用的特征时间的大小关系决定的</font>
+ 弹性回跳原理的主要论点
    - 造成构造地震的岩石体破裂是由于岩石体周围地壳的相对位移产生的应变超过岩石强度的结果 -> 应变触发断裂
    - 这种相对位移不是在破裂时突然产生的，而是在一个比较长的时期内逐渐达到其最大值的 -> 应变需要积累
    - 地震时发生的唯一物质移动是破裂面两边的物质向减少弹性应变的方向突然发生弹性回跳。这种移动随着破裂面的距离增大二逐渐衰减，通常延伸仅数千米 -> 断裂弹跳方向
    - 地震引起的震动源于破裂面，破裂表面一开始很小，很快地扩展非常大，但是其扩展速率不会超过岩石中P波传播的速度 -> 断裂过程
    - 地震时释放的能量在岩石破裂前十以弹性应变能的形式存储在岩石中的 -> 能量转化过程
+ 应力与应变
    - 受到应力时的相应：形变/流动/断裂
    - 应变：应变没有量纲，是相对变形的量
+ 地震能：大部分应变能转化为热能，只有百分之几的应变能转化为地震波的能量
    - 地震效率：地震波能量比上地震能，约为7.5%-15%
    - 应变能从重力和地球内部的热驱动着板块运动而来
+ 弹性回跳理论：只是一个假说模型，并不是真正的情况

### 第三节 震源机制解
+ 震源物理是指研究地震孕育/发生的物理过程及相关的物理现象
+ <font style="color:#8A8F8D;">震源物理的历史</font>
    - <font style="color:#8A8F8D;">1910年，里德提出弹性回跳学说</font>
    - <font style="color:#8A8F8D;">1923年，日本地震学家中野广首首先发现地震记录的地面初动四象限分布，并由此发展处地震真元的无矩双力偶点源模型</font>
    - <font style="color:#8A8F8D;">20世纪五十年代苏联科学家提出地震震源的等效位错理论</font>
+ <font style="color:#8A8F8D;">震源模型分类</font>
    - <font style="color:#8A8F8D;">点源模型：单力偶震源模型、双力偶震源模型</font>
    - <font style="color:#8A8F8D;">非点源模型：有限移动震源模型、位错震源模型</font>
+ <font style="color:#8A8F8D;">震源模型提供的两组力学参数</font>
    - <font style="color:#8A8F8D;">断层面走向、倾向和倾角</font>
    - <font style="color:#8A8F8D;">最大主应力轴、最小主应力轴和中等主应力轴的方位和产状</font>
+ 震源机制解（又称地震机理）：指断层方位、位移和应力释放模式以及产生地震波的动力学过程
    - 震源辐射图案（地震小球）：挤压部分（P）是黑的，拉伸（T）的是白的
    - 震源球
    - 确定断层面的方法：极震区等阵线长轴的方向；大地测量数据；前阵以及余震的分布；S波偏振方向；有限移动源位移谱
+ 断层越陡正断层越容易发生，断层越缓逆断层越容易发生

### 第四节 板块构造学说
+ 板块理论发展的三个阶段：大陆漂移、海底扩张、板块构造
+ 原来的观点：固定论 -> 没有主动的水平运动，竖直运动带动水平运动，褶皱的形成
+ 板块：刚性（或半刚性的）固态的巨大板状岩石块体
    - 板块构造：地球最外层由若干个大小不等的板块组成，它们漂浮在相对较软的热物质上作相互运动
    - 软的部分 -> 软流层；类似于润滑作用
+ 灾变论与均变论的历史
    - 17世纪以前，地质现象的解释是灾变说（即质变）
    - 1785年，苏格兰地质学家James Hotton提出均变原理（即量变）
    - 19世纪中叶，均变说被地址学家广泛接受，“将今论古方法”称为地址学中最重要的原理
    - 直到19世纪末20世纪初，权威的地质学说认为，地质构造的运动主要是垂直向的，横向运动只是垂直运动的副产品
    - 最终灾变论被认为是占主导的，均变论并不是主导的
+ 槽台说：地台是稳定的，地槽是活动的，海洋总归是海洋大陆总归是大陆，若有变化也不是原地升升降降而已
+ 垂直运动说面临的困境
    - 阿尔卑斯-喜马拉雅大规模推覆构造
    - 太平洋西部的弧状岛屿、岛弧后的边缘海盆地
    - 恐龙的化石几乎遍布世界各大陆
    - 南极大陆储藏着煤
    - 酷热的非洲大陆有冰川的遗迹
    - **<font style="color:#DF2A3F;background-color:#FBDE28;">1915年德国气象学家魏格纳提出“大陆漂移假说”</font>**
+ 艰难考证大陆漂移的证据
    - 形状复杂的板块拟合 ->运用大陆架后板块接合比较好
    - 化石 -> 中龙属、水龙兽、犬颌兽属等等
    - 古气候数据 -> 现今的气候区发现了不对应的气候现象
+ 板块学说发展的第二阶段：海底扩张（洋中脊记录地磁信息，地震和火山打都分布在海沟和海岭）
    - 岩石的两种分类：正常极性和反常极性（按照磁极方向与当前地磁场磁极方向）
    - 居里温度：580摄氏度，高于这个温度时会失去磁性
    - 洋中脊形成不会引起过多褶皱最初的解释：地球膨胀假说 -> 没有证据支持
    - 洋壳的更新与循环：洋中脊处产生，海沟处消亡
+ **<font style="color:#DF2A3F;background-color:#FBDE28;">1962年Hess正式发表论文"History of Ocean Basins"</font>**
+ 板块学说的第三阶段：板块构造
    - 地球结构：地壳、地幔、地核
    - 七大板块：印度-澳大利亚板块、太平洋板块、欧亚板块、非洲板块、南极洲板块、北美板块、南美板块
    - 板块边界的类型：扩散边界、汇聚边界、转换边界、板块边界区域
        * 板块边界区域：有一定宽度的带，没有清晰的板块边界，相互作用不清楚
        * 所有版块的边界都不清晰！
        * 扩散边界：地壳产生的地方，例如大洋中脊
        * 汇聚边界：地壳消亡的地方
            + 三种碰撞：海洋与陆地岩石圈的碰撞（形成海沟），海洋与海洋岩石圈的碰撞，陆地与陆地岩石圈的碰撞（形成山脉）

+ 板块理论的地震学证据
    - 天然地震的空间位置，可以勾画出板块的边缘
    - 震源机制解，可以确定板块活动的力学性质
    - 地震测深所得深度速度分布，可以给出板块运动的物理条件 -> 软流层软，地震波传播速度慢，也让板块移动提成为可能
+ 板块理论的震源机制解的证据
+ 板块理论的深部速度结构的证据
    - 软流层是一个低速层，同时让板块的移动成为可能

### 第五节 全球地震活动情况
+ 全球地震活动概况
    - 地震的空间分布显示出地壳和上地幔的不均匀性，而地震的时间序列和发生规律又能反映地壳应力和介质的某种力学性质
    - 地震的几个要素包括发生地点、发生时间和释放的能量大小等 -> 一般来说大地震都是逆断层引起的
    - 有地震仪器记录以来全所发生的特大地震全都发生在海洋板块向大陆板块的俯冲的边界带上

$ \lg E = 11.8 + 1.5M $

+ 其中E为能量，单位为尔格，大小为10^-7焦耳；M为震级
+ 全球的地震带有三个：环太平洋地震带、阿尔卑斯-喜马拉雅地震带（即欧亚地震带）和海岭（大洋中脊）地震带

### 第六节 不同类型的地震
+ 不同地震类型
    - 地面震动：天然地震、人工地震与脉动
    - 天然地震：构造地震（地下岩层错动而破裂缩成（90%）），火山地震（7%），陷落地震（3%）
        * 地面发生地震比较频繁的原因：冷而脆
    - 人工地震：包括工业爆破、地下核爆炸引起的振动、水库地震等等
        * 为什么会产生水库地震？-> 因为认为增加了压强，原来没有达到临界值的岩层可能会到达临界值发生断裂引起地震
        * 鉴别地下河实验的地震学方法
            + 震源深度不会太深
            + 震中位置一般在荒漠
            + 核爆炸的波形不如地震的复杂
            + 核爆炸P波成分高，S波成分很低；但是天然地震S波成分显著而且高于P
            + 核爆炸初动解Beachball全黑，因为都是冲击；对应的塌方事件Beachball全白，因为类似于是一个逆过程
            + 地震波频谱
            + 体波和面波震级比不同，面波强度相同时核爆炸的体波强度大
+ **<font style="color:#DF2A3F;background-color:#FBDE28;">全球地震分布是比较均衡的，南半球的地震并不比北半球少，但是北半球陆地（1）面积大、（2）人口多，（3）而且北半球地震带大多分布在陆地上，那么北半球地震对人类的影响就大，人类也就更加关注，造成了北半球地震比南半球多的假象</font>**
