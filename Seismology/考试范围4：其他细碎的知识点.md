### 数据
+ 全球每年约发生500万次地震，人们可以感觉出来的仅占1%，造成严重破坏的7级以上大地震越有18次，8级以上的特大地震1-2次。
+ 全世界越有6亿多人生活在强震带上，20世纪越有200万人死于地震，预计21世纪将有约1500万人死于地震
+ 20世纪以来，我国发生了800多次6级以上的地震，平均每年约8次
+ 历史记载全球死亡超过20万人的地震有6次，其中在中国就有4次
+ 浅源地震中源地震深源地震分界，60km和300km
+ 地方震近震远震分界，100km和1000km
+ 弱震震级(-∞,3)，有感地震震级[3,4.5]，中强震震级(4.5,6)，强震震级[6,+∞)，巨大地震[8,+∞)
+ 地表附近的岩石泊松比接近1/4，P波是S波速度的根号3倍
+ 洛夫波和瑞利波比P波和S波持续的时间长5倍多
+ 大陆地区，地壳平均厚度为35km，青藏高原60-80km，华北地区不到30km，海洋地壳的厚度只有5-8km
+ 上中下地幔的分界线，410km和670km
+ 120度，延迟时间10分钟
+ 地震效率的数值在7.5%-15%
+ 平均误差88km
+ 海洋的平均深度约为4.5km
+ 海岭不是永久的形态，它的寿命不超过两三亿年
+ 地下100-200km，软流层
+ 洋中脊的板块厚度只有0-10km，南美洲板块的厚度可达200-300km
+ 大约全球80%的浅源地震、90%的中源地震和全部的深震都集中在环太平洋地震带上
+ 名称：阿尔卑斯-喜马拉雅地震带，与初中高中叫法不同；另外还有海岭地震带
+ 构造地震90%，火山地震7%，水库地震3%
+ 水库地震的震中仅分布在水库及其周围，一般位于水库及其5km的范围内，少有超过10km
+ 核爆炸所激发的地震波较相同大小的天然地震所激发出来的地震波包含更大比例的高频成分
+ 震中定位的精度大概是10km，震源深度的精度更差，大约为20km
+ 仅从1970到1981年期间，在死于自然灾害的113万人中，死于地震者44万，
+ 地震预测长期（十年以上）、中期（1-10年）、短期（10-数百日）、临震（10日以下）
+ 氡浓度异常所及的范围，一般距离震中200-300km，有时到600km。浓度变化幅度为20-100km。
+ 一般由地震引起的磁偏角和倾角范围在10分以内
+ 破坏性地震从人感觉震动到建筑物破坏平均只有12秒钟
+ 日本使用气度烈度表，我国使用十二度烈度表，与国际惯例一致
+ 《中国地震烈度区划图》1974-2073最大地震烈度图，比例三百万分之一
+ 自大庆油田开发以来，95%的新油田都是地震勘探提供构造的
+ 检波器距离一般是25或者50米，被称为道间距
+ 目前世界上钻井最深的记录为20世纪70年代苏联人所创造的，它们在俄罗斯莫曼斯克州得科拉半岛上完成了深达12km的超深井
+ 海啸。三种运动方式中逆冲断层垂直方向上运动最大，正断层其次，走滑断层几乎没有垂直运动
+ 远洋海啸波是一种长波，波长可达几百千米，周期接近一个小时
+ 近海海啸到达海岸的时间也较短，只有几分钟，多者几十分钟，海啸预警时间则更短或者根本无预警时间，很难防御
+ 全球有记载的破坏性较大的的海啸约发生260次，平均六七年发生一次，其中发生在环太平洋上的海啸约占80%，发生在地中海区的约占8%，而在日本及其临近还与发生的海啸则占太平洋地区的60%左右
+ 公元684-1983年间，日本共发生62次比较严重的海啸。
+ 两千年以来，中国只发生过10次海啸，平均200年左右才出现一次
+ 中国海区处在宽广的大陆架上，渤海平均深度约为20m，黄海平均深度约为40m，东海平均深度约为340m。
+ 还笑波的速度仅为几百千米每小时



### 人、时间和日期
+ 伊壁鸠鲁认为地震是由于风被封闭在地壳里；卢克莱修认为来自外界或打的本身的风和空气的某种巨大力量突然进入大地的空虚处；亚里士多德认为“气”是地震的成因
+ 前1831年，泰山震，可能是最早的地震文字记载
+ 前12世纪，《吕氏春秋》记录“周文王八年地动”
+ 132年，张衡创制候风地动仪
+ 138年，中国东汉张衡在洛阳设置额一台候风地动仪检测到了一次发生在甘肃省内的地震，这时人类历史上第一次用地震仪器检测到地震
+ 1057年，固安6.75级地震
+ 1522年9月6日，麦哲伦船队完成人类历史上第一次环球航行
+ 1556年陕西关中大地震，历史上死亡人数最大的地震
+ 1564年，意大利出现人类历史上最早的烈度表
+ 1570年11月意大利佛拉拉发生大地震
+ 1571年，加里西奥总结地震时提出地震前兆现象，并建议了预防地震的若干措施
+ 1624年，滦县6.25级地震
+ 1666年，牛顿发现万有引力定律；牛顿和惠更斯同时得出地球是一个两级扁平赤道隆起的椭球的理论
+ 1679年，三河平谷8级地震，毁坏了北京的皇宫和白塔
+ 1755年11月1日，万圣节，距离葡萄牙里斯本城几十千米的大西洋海底发生强烈地震
+ 1798年，英国的卡文迪许确定地球的平均密度大约是5.45克每立方厘米
+ 1845年，英国地震学家马利特利用人工激发的地震波来测量地壳中弹性波的传播速度
+ 1875年，爱尔兰工程师罗伯特马莱，在意大利研制了第一台有事件记录的地震仪
+ 1883年，印尼附近发生的海啸属于火山海啸
+ 1885年，英国物理学家瑞利首先在理论上导出
+ 1887年，地球物理学家乔治·达尔文得出结论，认为地球内部是流体的假说不可取
+ 1889年，帕斯维奇在德国波茨坦第一次记录到远震
+ 1892年，约翰米尔恩、詹姆尤因、托马斯格雷研制出记录地震动岁时间变化质量较高的地震仪器
+ 1895年，米尔恩回到英国，在怀特岛的赛德建立了地震台
+ 1896年6月15日，日本东部发生8.5级大地震，紧接着发生异常退潮，史称“明治海啸”
+ 1897年，加州的地震台装配的就是这种地震仪
+ 1897年，维歇特通过理论计算发现，地球内部可能由一个围绕着铁核的硅酸盐地幔组成
+ 1898年，德国地震学家维歇特第一台有黏滞阻力的地震针
+ 1900年初，俄国伽利津王子研制出来了第一台电磁式地震仪
+ 19世纪末，英国地震学家奥尔德姆画出最早的走时表
+ 1900年，奥尔德姆在地震图上识别出P波、S波和面波
+ 1902年，柏林发表了地球内部略图，地球的早期模型有固体地壳、弹性地幔和固态核
+ 1906年4月18日，美国旧金山8.3级地震
+ 1906年，日本地震学家今村明恒在他所写的一篇论文中曾确认东京近海的相模湾为地震空区，成功预报了1923年东京大地震
+ 1906年，奥尔德姆根据震源—接收器距离约100度时没有直达的P波、S波的观测事实发现了地核的存在
+ 1908年12月28日，意大利墨西拿7.5级地震，引发海啸
+ 1909年10月8日，克罗地亚大地震
+ 1910年，美国地震学家里德提出弹性回跳理论
+ 1910年，南斯拉夫地震学家莫霍洛维奇发现莫霍不连续面
+ 1911年，英国力学家洛夫首先提出洛夫波
+ 1914年，德国地震学家古登堡发现在地下2900km深处的物质分界面，即古登堡面
+ 1912年，德国天文、气象、地球物理学家魏格纳第一个对大陆漂移学说作系统论述
+ 1915年，魏格纳在负伤休假期间，出版著作《海陆起源》，更系统地阐述了大陆漂移的理论
+ 1919年，德国明托普获得折射初至法专利
+ 1920年，甘肃大地震
+ 1920年12月16日，中国海源8.5级大地震
+ 1923年9月1日，日本东京8.2级大地震
+ 1924年，中野广首首先发现地面初动向四象限分布（1923？）
+ 1930年，地震学家李善邦和秦馨菱先生北京西山鹫峰观测太，中国自建的第一个地震台
+ 1931年，日本学者提出岩浆冲击说
+ 1931年，日本地震学者末广恭二赴美国讲学，讲题为“工程地震学”
+ 1933年，日本三陆地区发生8.1级大地震，正断层地震
+ 1935年，在加州理工学院，里查德对于南加利福尼亚地震提出震级标度，称为里氏震级，是第一个被广泛使用测量地震大小的标度
+ 1936年，新西兰学者提出地震成因的相变理论
+ 1936年，莱曼发现了固体内核
+ 1938年，拜尔利第一次提出震源断层面解的方法
+ 1939年，杰弗里斯和布伦合编的走时表（简称J-B表）和古登堡走时表最为完整，它们基本上是相同的
+ 20世纪40年代，哈瑞提出的共深度点记录反射波的思想在如今的野外勘探中已经普及
+ 20世纪50年代前苏联科学家提出了地震震源的等效位错理论
+ 1950年，英国海洋学家朗格特-希金斯发展一项理论将气旋理论和海岸理论统一起来
+ 1952年11月4日，勘察加大地震
+ 1954年，美国地震学家贝尼奥夫对许多岛弧上的地震分布和地震深度进行了系统的研究，发现有一条以45度角向大陆方向倾斜的深源带，这就是人们所成的贝尼奥夫带，海洋板块在这里消亡。
+ 1956年，中国出版了两卷《中国地震资料年表》
+ 1956年，第一届世界地震工程会议在美国举行
+ 1959年，中国王振铎复原了候风地动仪
+ 20世纪60年代初期，Hess和Dietz分别提出了一个海底扩张学说
+ 1960年5月22日，智利大地震，引发火山喷发，有异常退潮现象，属于地震海啸中的“下降型”海啸，22小时后到达日本海岸
+ 1962年，梅德韦杰夫著有《工程地震学》一书
+ 1962年3月19日，新丰江6.1级地震，我国最大的水库地震，之前弱震b值下降
+ 1963年，瓦因和马修斯提出一个学说，海洋地壳的第三层是软流层上升的物质由海岭涌出后向两边扩张所形成的
+ 1964年，考克斯等人发现了300多万年以来，地磁场曾经三次转向
+ 1964年，梅德韦杰夫、施蓬霍伊尔、卡尔尼克创立国际烈度表，即MSK烈度表
+ 1964年，阿拉斯加一带发生了9.2级地震
+ 1965年，26个国家和地区进行合作，在夏威夷建立了太平洋海啸劲爆中心（PWTC）
+ 1966年，邢台7.2级地震
+ 1966年，安艺敬一提出地震矩，物理基础更加明确
+ 1968年，美国重新编制了全球平均的P波走时表
+ 1968年，勒比雄将全球岩石圈分成6个大板块，分别是欧亚板块、美洲板块、非洲板块、太平洋板块、印澳板块和南极板块
+ 1969年7月18日，渤海中部发生7.4级地震，引发海啸，给河北唐山地区造成一定危害，我国近海检测记录到的三次海啸之一
+ 20世纪70年代中期，以前的地震仪基本上都是模拟记录地震仪，以后地震观测系统采用了将接收信号数字化后进行记录的方式
+ 1970年5月31日，秘鲁钦博特7.8级大地震
+ 1976年7月27日，中国唐山7.8级地震
+ 1980年，国际地球标准模型委员会推荐杰旺斯基和安德森教授提出的初步地球参考模型（PREM）
+ 1981年，第21届国际地震学和地球内部物理学委员会（IASPEI）正式通过PREM
+ 1983年，中国加入太平洋海啸警报中心，随之国家海洋局海洋环境预报中心开展了海啸警报业务
+ 1983年，日本海附近发生的海啸之前有异常涨潮现象，属于地震海啸中的“隆起型”海啸
+ 1983-1986年中国地震局与美国地质调查局合作建设中国数字地震台网（CDSN）
+ 1984年，美国57所大学联合成立了一个叫做IRIS的联合体，计划在全球设置100多个数字地震台
+ 1985年9月19日，墨西哥8.1级大地震
+ 1989年10月17日，美国洛马普瑞特7.1级大地震，是自1906年旧金山大地震后袭击旧金山海湾地区最大地震
+ 1992年1月4日至5日，海南岛南段发生海啸，给当地造成了一定的损失
+ 1994年，台湾海峡发生海啸，不过这次海啸并没有造成损失
+ 1995年1月16日，日本神户7.2级大地震
+ 1996年，Robert Geller等人在国际权威学术刊物《自然》和《科学》等杂志连续发表文章，提出地震不能预报，引起威斯等人针锋相对地发表反对文章，引起激烈争论
+ 1998年，巴布亚新几内亚太平洋海岸发生的海啸属于滑坡海啸
+ 1999年9月20日，中国台湾7.6级大地震；1921年9月21日集集大地震
+ 20世纪90年代后期，国家海洋局组织开发了太平洋海啸资料数据库、太平洋海啸传播时间数值预报模式和越洋、近海海啸数值预报模式。
+ 2001年1月26日，印度古吉拉特邦7.9级大地震
+ 2004年12月26日，印尼苏门答腊发生震级9.0的特大地震，引起海啸造成严重灾难
+ 2008年5月12日，中国汶川8.0级大地震
+ 2011年，日本仙台大地震
