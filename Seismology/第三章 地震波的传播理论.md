### 第一节 地震波传播的基本概念
+ 地震波传播计算：研究时假定岩石为一种完全弹性体
+ 地震波的分类，按照传播路径
    - 直达波：震源->上层介质->台站
    - 反射波：震源->上层介质->界面（立即折返）->上层介质->台站
    - 直达波：震源->上层介质->界面（走一段距离）->上层介质->台站
        * 又称侧面波、折射波、衍射波，行走反射波等等
        * P波和S波都会有相应的首波，S波传播速度较慢
        * 超越（第一）临界震中距时会有首波出现，超越第二临界震中距时首波在到时最短
+ 地震波的吸收和衰减
    - 振幅指数衰减，随时间衰减系数γ，随距离衰减系数

$ A = A_0e^{-\gamma t} $

$ A = A_0e^{-ax} $

    - 品质因子Q：一个周期后损失的能量占比的倒数【联系普物】

$ \frac{1}{Q} = \frac{1}{2\pi}\cdot\frac{\delta E}{E} $

+ 震中距：震中到观测台站的距离
    - 单位制1：千米
    - 单位制2：度（大约相当于110km）

### 第二节 地震波传播的基本理论
+ 研究地震波在地球内部传播的问题的两种方法
    - 动力学方法：直接求解波动方程【类比波动光学】
    - 运动学方法：将求解波动方程求解转化为波传播的涉嫌理论，用地震射线这一概念，研究地震波在地球内部传播的运动学特征【类比几何光学】 -> 射线地震学/几何地震学
+ 费马原理（Fermat's principle）：地震波的高频近似解 -> 衍射现象不明显
    - 原文
        * 光学版本：QP两点间光线的实际路径，就是光程（QP）（或者说所需的传播时间τ）为平稳的路径（摘自赵凯华、钟锡华《光学》）
        * 地震学版本：QP两点间地震射线的实际路径，就是地震波传播时间为平稳的路径
    - 对于“平稳”的准确理解
        * 在微分学中说一个y=f(x)函数在某处平稳，是指它的一阶微分$ dy=0 $，在这里可能是极小值（$ d^2y>0 $），可能是极大值（$ d^2y<0 $），也有可能是拐点、常数等等其他情况
        * 费马原理中“平稳”的含义就是光线在实际路径上的光程变分为0
    - 对于“平稳”的粗略理解：时间最短的传播距离

<img src="https://cdn.nlark.com/yuque/0/2024/png/2371504/1721461572154-f75e724e-bf58-48f8-ba45-5d30b86acc32.png" width="250" title="" crop="0,0,1,1" id="u91a5936a" class="ne-image">

+ 斯涅耳定律（Snell's Law）：费马原理在反射、折射中的应用
    - 第一定律：反射定律（入射角反射角相等）
    - 第二定律：折射定律（波速、正弦关系）
        * 射线参数：界面处传播方向与法线夹角正弦值与波速的关系
            + 对于给定的射线（频率固定），射线参数是一个常数，在传播过程中不变
            + 近震情况下，地球曲率对传播时间的影响可以忽略<img src="https://cdn.nlark.com/yuque/0/2024/png/2371504/1721463958790-5ee89f6d-5025-4807-8c81-c9564ad11bda.png" width="342.9965515136719" title="" crop="0,0,1,1" id="LAgXH" class="ne-image">
            + 远震情况下，地球曲率对传播时间的影响不可以忽略

        * 临界透射：波从波疏介质射向波密介质，正好发生全反射时的现象
+ 地震波的转换
    - P波、SV波入射：可能产生反射P波、折射P波、反射SV波、折射SV波
    - SH波入射：不会发生转换

+ 主要震相特征
    - P：在震中距为100度的范围内，P将作为地震记录的第一个震相清晰地显示出来。一超过103度，其振幅就变小，这是因为进入地核的阴影区所致。当看到弱小的波时，一般认为那是在核幔边界上由于衍射而产生的，这类似于莫霍面衍射的Pn波。
    - S：在震中距最大为100度的范围内，S往往以比P还大的振幅在地震记录上显示出来。超过100度时，虽然开始进入了地核隐区。
    - PP、SS：这两个震相在震中距超过20度是就开始与P或S分离。
    - pP、sS：当发生深震时，在30-100度附近，在P、S之后可以清晰的显示出来。pP和P的到时差，以及sS和S的到时差，往往随着震源深度不同而差别很大，因此对确定震源深度非常有用。
    - PcP、ScS或者是PcS、ScP常在震中距在30-40度左右显示出来。
+ 地震走时表：地震波在不同震中距上传播的时间表
