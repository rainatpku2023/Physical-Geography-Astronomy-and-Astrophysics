### 地图投影的基本问题
+ 地图投影的定义：按照一定的数学法则，将地球椭球面的经纬网转换到平面上建立地球表面上地理坐标（φ，λ）与地图上对应的平面直角坐标（x，y）之间一一对应的函数关系。
+ 地图投影的基本问题：投影变形，每一张地图都会有失真变形
    - 一般而言，在某种性质的投影中，不同位置上的投影变形大小不一，而且甚至同一点的不同方向上的变形也是不一样的。

### 地图投影理论
+ 变形性质分类
    - 等角投影：用于航海、洋流和风向图等
    - 等面积投影：适合于对面积要求较高的自然社会经济地图
    - 任意投影：角度变形小于等面积投影，面积变形小于等角投影
        * 常见的如等距投影，该投影在经线方向不变形，沿着经线的方向m=1。

+ 构成方法
    - 几何投影：圆柱投影，圆锥投影，方位投影
    - 非几何投影（即条件投影）：正轴、斜轴和横轴投影，切投影和割投影；伪圆柱投影，伪圆锥投影，伪方位投影
+ 变形椭圆：Tissot椭圆（微分椭圆）
    - 主方向长度比：最大长度比a，最小长度比b，经线长度比m，纬线长度比n，最大角度变形ω
    - 面积比，面积变形：面积变形为面积比与1的差。
    - 角度变形：地面上某一角度投影之后角度与原来固有角度的差距。一点上角度变形是不同的。因此在讨论某一点的角度变形时，只研究其中最大的角度变形值。
+ 按照研究区域划分：
    - 小区域地图常用投影：圆锥投影，高斯克吕格投影，UTM投影
    - 半球区域地图常用投影：方位投影
    - 全球区域地图常用投影：圆柱投影

+ 墨卡托投影
    - 正轴等角切圆柱投影
    - 没有角度变形
    - 有面积变形，而且从赤道向两极不断增大
+ Web墨卡托
    - 将地球看作正球体
    - 注意，导航计算时不能直接在地图上测量，需要实地测量或者考虑投影（考虑实际经纬度数据）
    - 网络上的地图瓦片只是给人一个直观的表现
    - 街区情况基本上不改变，地图上量测也基本准确，从全球到微小地区都能无缝切换
+ 高尔-彼得斯投影
    - 等积圆柱投影
+ 罗宾森投影
    - 矫正法投影，形状、距离、方向都有投影，但是每个都没有那么大
+ 贝尔曼圆柱投影
    - 等积投影，两极区域东西方向拉长，赤道区域南北方向拉长
+ 温克尔投影
    - 1998年，美国国家地理学会将罗宾森投影换成温克尔投影，之后许多教育机构也选择温克尔投影作为世界地图
+ 兰伯特投影
    - 正轴等角投影

+ 高斯-克吕格投影
    - 等角横轴切（椭）圆柱投影
    - 中央经线上没有长度变形，同一条经线上纬度越低变形越大，最大变形在赤道上。同一条纬线上距离中央经线越远变形越大。
    - 可以应用于国家基本比例尺地形图（国家地图）
    - 6度带第1带的中央经线为东经3度，3度带第1带的中央经线为东经3度
    - 1:2.5万-1：50万用6度分带投影，1:1万及更大比例尺（更小范围）用三度分带投影。

dx为投影平面上的纵向线段长度，Rdφ为大地地理坐标系南北方向上的微分圆弧长度。

dy为投影平面上的横向线段长度，dy = αdλ，Rcosφdλ为大地坐标系上东西方向上的微分圆弧长度。

等角条件为mn相等，即两个主方向（正好是经向和纬向）长度变形率相等。

+ UTM投影：注意东经和西经的原点和方向，UTM是6度分带

### 地图投影选择的影响因素
+ 制图区域大小、形状和地理位置的影响
    - 投影的无变形点和线应该位于制图区域的中心位置，等变形线应该与制图区域的形状一致，从而保证制图区域的变形分布均匀
        * 极地位置：正轴方位投影
        * 赤道附近：横轴方位投影正轴圆柱投影
        * 半球地图：横轴方位投影
        * 中纬地区：正轴圆锥投影或者斜轴方位投影
        * 低纬度地区：正轴圆柱投影

### 数据中的问题
火星坐标系。个人上传数据要转换到火星坐标系，个人下载数据也要从火星坐标系中转换回来。

只有天地图时没有纠偏的。

### 三北指针
+ 真北：经线的北方向
+ 坐标北：即坐标纵线靠近北方的方向
+ 磁北：磁北极方向
+ 磁偏角：磁子午线和真子午线之间的夹角
+ 子午线收敛角：坐标纵线和真子午线之间的夹角。
+ 磁坐偏角：磁子午线与坐标纵线之间的夹角。

### 不同椭球之间的坐标转换
+ 四参数转换
    - 地理坐标所对应的**高斯平面直角坐标**转换为另一个坐标系的**高斯平面直角坐标**
    - 两个平移参数、一个旋转参数、一个尺度参数
+ 七参数转换
    - 三个平移参数、三个旋转坐标，一个尺度参数
    - 需要3个控制点
+ 三参数转换
    - 区域范围不大，可以直接平移，XYZ旋转和尺度变化m都视为0。
+ 空间校正：
    - 提供用于对齐和整合数据的交互式方法。它通常用于已从其他源（例如 CAD 绘图）导入数据的场合。
+ 地理配准：
    - 使用地图坐标为地图要素指定空间位置。
