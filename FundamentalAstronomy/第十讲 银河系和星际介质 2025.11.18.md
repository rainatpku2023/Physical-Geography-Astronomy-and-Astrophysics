### 银河系
#### 银河系概况
+ 对银河尺度的认识：原来缺乏星际消光的认识，所以估计值偏大。
+ 银河系的结构：
    - 英仙臂(Perseus)
    - 船底 (Carina) –人马(Sagittarius) 臂
    - 半人马臂
    - 天鹅臂(Cygnus)
    - 猎户臂(Orion)
+ 银河系的成分：银盘、核球、银晕

#### 星族
+ 星族I恒星
    - 年轻的、富金属恒星（金属丰度为太阳值的0.1-2.5倍）主要位于银盘中，绕银心作圆轨道运动；如疏散星团（为何年轻的恒星富金属？）
    - 银盘是气体富集地，也是恒星形成的富集地，所以聚集了各种年龄的恒星
+ 星族II恒星
    - 年老的、贫金属恒星，主要位于银晕和核球中，以银心作为中心球对称分布绕银心作无规则的椭圆轨道运动；如球状星团
    - 银晕中几乎没有气体和尘埃，也就没有恒星的诞生；恒星形成早在100亿年前就已经结束

<img src="https://cdn.nlark.com/yuque/0/2025/png/2371504/1763471799404-aea6bfa1-4cf6-418c-9ded-1cc2e796e1bb.png" width="482.6666666666667" title="星族I（A）和星族II（B）的运动轨道" crop="0,0,1,1" id="ua16a64ee" class="ne-image">

+ 星系盘内的星族I恒星绕银心作规则的圆轨道运动
+ 晕中的星族II恒星绕银心作大偏心率的椭圆轨道运动，且轨道取向随机

#### 银盘
+ 构成：星族I恒星，气体和尘埃。
+ 直径：30kpc
+ 薄盘厚度：300pc
+ 观测方式：
    - 光学观测：
        * 示踪天体：OB型星、年轻的疏散星团、发射星云和HII区、经典造父变星
        * 方法：视星等＋绝对星等（标准烛光）、距离模数 m-M → 距离、造父变星周光关系 → 距离
        * 限制：星际尘埃消光
    - 红外观测：不同银经方向，视线方向穿过不同的旋臂，恒星数密度分布出现差异。
    - 射电观测：
        * 示踪天体：HI区、分子云
        * 方法：测量HI区 21cm谱线 和分子云的毫米波谱线的Doppler谱线位移→ 视向速度 → 转动速度、比较银河系自转曲线→ 距离
        * 限制：气体云的转动是非圆的，在圆运动的同时还有无规运动

#### 旋臂
+ 旋臂不是物质臂
+ 密度波理论：天体还是按照原来的轨道运动，只不顾分布形成了一个密度波；有点类似于群速度和相速度。
    - 星系引力势扰动
        * →银盘内的天体以椭圆轨道运动
        * →运动速度变化
    - 轨道取向相互耦合
        * →物质密度的规则变化
        * →密度波

#### 核球
+ 以银心为中心，呈现拖球形，大小大概6×4kpc
+ 恒星分布十分密集，数密度～1,600 ly-3, 是银河系平均恒星密度的105倍
+ 恒星颜色黄-白（年龄介于盘和晕之间）
+ 有气体和尘埃，尤其是其内部；
+ 内部有恒星形成；比较随机或略带旋转的运动轨道；中心可能有棒状和环状结构
+ 光学观测
    - 在光学波段，核球附近区域受星际气体和尘埃的遮挡
    - 辐射主要来自年老的星族I天体（如红巨星）

#### 银晕
+ 近似球状分布，或者略扁
+ 只有年老的星族 II；几乎没有气体和尘埃；过去100亿年中几乎没有恒星形成
+ 恒星运动轨道随机
+ 无明显子结构

#### 暗物质
+ 由银河系旋转曲线得知，银晕中不可见物质质量远远超过可见物质质量（若无暗物质晕，旋臂不能长时间稳定）
+ 暗物质的特征：在所有波段都不产生辐射，仅有引力

#### 银心
+ 受消光影响，在可见光波段看不到中心，需要红外、射电或者 X-射线波段
+ 由银心附近恒星的运动推测在银心集中了～4×106 M⊙的质量
+ 人马座(Sagittarius)A的尺度 < 10 AU → 超大质量的黑洞

### 星际介质
#### 星际介质
+ 银河系星际空间平均0.5个原子/cm3；地球大气约2.4×1019 个分子/cm3
+ 星际介质（ISM，Interstellar Medium）：星系内分布在恒星与恒星之间(~6-10 ly)的物质
    - 质量大概占恒星质量的10%
    - 星际介质包括：星际气体、星际尘埃

#### 星际气体
+ 成分：
    - 星际气体主要由H构成（90% 按数密度）
    - 其次为He（~9%）
    - 以及少量金属元素
+ 按不同环境下H的存在方式不同，可以分为：
    - 电离气体：最热
        * HII区：被高温（O, B0-2型）恒星的紫外辐射激发或电离的星际物质，也称为HII区。
        * 紫外和X射线观测还发现存在一类温度高达106-107K，典型密度10-3 cm-3 的热气体,称为云际冕气(coronal gas) 。
        * 20 - 60%的星际空间被云际冕气占据。 这些气体的加热和电离主要来自超新星爆发和星风的激波。
    - 中性原子气体：中等
        * 中性氢21cm超精细结构谱线成为我们研究星系中中性原子气体的主要探针。
    - 分子气体：最冷
        * 20 - 60%的星际空间被云际冕气占据。 这些气体的加热和电离主要来自超新星爆发和星风的激波。
        * 对于分子云典型的温度和密度条件，分子CO很容易被激发，转动能级跃迁谱线成为观测分子云的主要手段，12CO的辐射一般是光厚的，因此，通常观测13CO（光薄）来直接测量分子气体的柱密度。
            + 碳13的二氧化碳的转动能谱比更大量的碳12的二氧化碳转动能谱刚好错开一点，所以能够传出来。

#### 星际尘埃
+ 星际尘埃的光学作用：
    - 星际消光 (interstellar extinction): 星际尘埃对星光的吸收和散射造成星光强度的减弱。
    - 星际红化 (interstellar reddening): 星际尘埃对星光的散射随波长的变化而不同,对蓝光散射较多而对红光散射较少,因而造成星光颜色偏红。
+ 星际尘埃的性质
    - 成分：硅（silicate）或石墨（graphite）微粒, 外面被冰或二氧化碳包裹。
    - 形成： 形成于红(超)巨星的外层大气(低温→气体凝结),在恒星演化晚期被吹向星际空间。
    - 星际尘埃与星际分子的关系：分子云中的尘埃屏蔽了星光中的紫外线,使分子免遭瓦解, 尘埃有利于分子形成。

    - 绝对的消光量：与视线前方消光物质总量有关。

#### 星际尘埃云的观测
+ 发射星云（emission nebulae）：主要包括大质量恒星周围的电离HII区、行星状星云等；
+ 反射星云 (reflection nebulae) : 星云通过尘埃反射附近的热星的星光而发光，颜色偏蓝（短波辐射更容易被散射）。
