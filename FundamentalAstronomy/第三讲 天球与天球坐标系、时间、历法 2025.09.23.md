+ 球面三角的基本知识
    - 大圆： 通过球心的平面与球面的交线，是直径最大的圆

#### 天球与天球坐标系
+ 天球： 以任意点为球心，任意长为半径，为研究天体 的位置和运动而引进的一个与人们直观感觉相符的假想圆球。  
    - 天体在天球上的位置**只反映天体视方向的投影**
    - 天球上任意两天体的**距离用其角距表示**
    - 地面上两平行方向指向天球同一点
    - 任意点为球心观测者“**由内向外**”看
+ 其他概念：
    - **天极(celestial pole)**P：过天球中心做一与地球自转轴平行的直线（天轴），它与天球相交的两点为天极。
        * 相当于把整个经纬网推到了无穷远处。
        * 直观：将北极点和南极点推到无穷远处，得到天极。
    - **天赤道(equator)**：过天球中心做一与天轴垂直的 平面，它与天球相交的大圆为天赤道。
        * 相当于把整个赤道推到了无穷远处。

<img src="https://cdn.nlark.com/yuque/0/2025/png/2371504/1758626506430-fd7d6dbc-26f0-49d5-98de-d99f5d4a1480.png" width="320.62499522231525" title="" crop="0,0,1,1" id="u941fc3ff" class="ne-image">

    - **天顶(zenith)**：Z过天球中心做一直线与观测点的铅垂线平行，交天球于两点，位于观测者头顶的一点称天顶。 
        * 直观：站在地上头顶正上方天球的位置。
    - **天底(nadir)**：Z'与天顶相对的另一交点为天底。
        * 直观：站在地上脚底正下方天球的位置，即地心的方向
    - **真地平(horizon)**：过天球中心做一与铅垂线 垂直的平面，与天球相交的大圆为真地平。 
        * 直观：就是地平线

<img src="https://cdn.nlark.com/yuque/0/2025/png/2371504/1758626581495-33fed57b-25bb-48c8-8893-7293ee4e9656.png" width="293.1249956320972" title="天顶与天底" crop="0,0,1,1" id="u9cd5bda9" class="ne-image">

    - **天子午圈**：**过天极和天顶的大圆。**
        * 直观：站在地上的正上方点和南北极点三个点所在的平面
        * 子午圈：直观上是子时和午时太阳的位置所在的赤经圈。
    - **四方点**：天子午圈与真地平相交的两点为南北点，（靠近北天极的为北点）天赤道与真地平相交的两点为东西点。
        * 直观：站在地面上对于东南西北的判断。
    - **卯酉圈**：过天顶和东西点所做的大圆。
        * 直观：与天子午圈、地平面垂直的面。

<img src="https://cdn.nlark.com/yuque/0/2025/png/2371504/1758626690886-db7e5a7f-1ee2-473e-b4d5-ed22ba43c7b1.png" width="306.8749954272062" title="天子午圈" crop="0,0,1,1" id="u1953e08b" class="ne-image">

    - **黄道(Ecliptic)**：过天球中心做一与地球公转轨道平面平行的平面为黄道面， 与天球相交的大圆为黄道。
        * 直观：黄道面就是太阳运动的平面。
    - **黄极**：黄道所对应的两个极点。

<img src="https://cdn.nlark.com/yuque/0/2025/png/2371504/1758626776742-e963ef8e-e9ed-41b9-ba1e-6abef4ef6a9c.png" width="247.49999631196266" title="" crop="0,0,1,1" id="u5bfe65b3" class="ne-image">

    -  二分点： 黄道与天赤道的两个交点
        * 总共有两个：春分点 (Vernal equinox)，秋分点(Autumn equinox)
        * 在短时间内，地球进动和章动不明显，可以认为天赤道相对于背景星空不动；黄道面也相对于背景星空不动，因此春分点和秋分点都是相对背景星空静止的点。
    -  二至点： 黄道上与二分点相距90度的另两个点。 
        * 总共有两个：夏至点 (Summer solstice)，冬至点 (Winter solstice)

### 坐标系统
#### 地平坐标系
+ 坐标架的参考物：
    - 基本点：天顶、天底
    - 基本圈：真地平
    - 原点：北点
+ 坐标描述：
    - 地平高度h：天顶距的余角
        * 希望地平高度越高越好，天顶距越小越好，这样大气层比较薄，大气消光比较弱容易观察
    - 方位角A： 由**北点**沿真地平顺时针度量到过天体的地平经圈（范围0-360度，北0东90南180西270）

#### 时角坐标系
+ 坐标架的参考物：
    - 原点：子午圈与天赤道南边的交点，即太阳正午时刻相同赤经的在赤道上的位置
    - 基本点：天极
    - 基本圈：天赤道
+ 赤纬δ：由天赤道沿过天体的赤经圈向两极方向度量（0度到±90度）
+ 时角t：由原点沿天赤道顺时针量至天体所在赤经圈（0 h – 24h)
    - 直观：对于某地的观测者来说如果一个远处恒星（视为在天球上不动）的时角是+1h，那么说明1个小时前它应该在上中天。即时角 = 当前时间 - 上中天时间
+ 特征：
    - 赤纬不随时间、地点的变化而变。
    - 时角随时间增加且具有地方性。

#### 赤道坐标系（最重要）
+ 原点：春分点(vernal equinox)  -黄道和天赤道的交点
    - 原点随着一天时间变化东升西落，每一个天体的位置和地球的自转就无关了
    - 引入了一个和天球一起作周日旋转（在地面观测者看来）的点作为零点。
    - 天体的位置和地球的自转无关，即不随时间和地点的变化而改变。
+ 赤纬δ(Dec: declination)：同时角坐标系
+ 赤经α(RA: right ascension) ： 由春分点沿天赤道逆时针方向量至天体所在赤经圈
+ **为什么赤道坐标系重要：**
    - **原点（春分点）在天球上位置不动。**
    - **地球绕着地轴旋转，基本圈赤道在天球上始终在一个圆线上。度量更加自然。**

#### 黄道坐标系
+ 坐标架的参考物：
    - 基本点：黄极
    - 基本圈：黄道
    - 原点：春分点
+ 度量：
    - 黄纬β：由黄道沿过天体的黄经圈向两边度量 （0度—±90度）
    - 黄经λ：由春分点（相对背景固定）沿黄道逆时针量至天体所在黄经圈（相对背景固定）与黄道的交点（0度–360度)
+ 性质：而且黄道坐标不随观测时间和观测地点而变。
+ 作用：采用黄道坐标系便于研究太阳系内各种天体的运动情况。

#### 银道坐标系
+ 坐标架的参考物：
    - 基本点：银极
    - 基本圈：银道面
    - 原点：银道与赤道的北交点（1958年前）  
+ 度量：
    - 银纬 b：由银道向北（南）度量（0度- ±90度）
    - 银经 l：由银河系中心逆时针量至天体所在银经圈 与银道的交点 （0 0 – 3600 )  
+ **作用：研究银河系的消光，银纬越高消光越少**

#### 天体的周日视运动
+ 不同纬度的天体视运动：
    - 极区： 在地球的南、北极地区看到所有天体都平行于地平圈做圆运动  
    - 赤道地区： 在地球赤道地区看到所有天体 垂直于平面做圆 运动  
    - 中纬度地区： 在中纬地区看到 的天体的周日视运动 
+ 天体的中天
    - 上中天：天极以南方向过子午圈（天极和天顶的大圆）
        * 对应太阳正午
    - 下中天：天极以北方向过子午圈（天极和天顶的大圆）
        * 对应太阳子夜
    - 如果上中天仍然在地下 -> 永不上升的天体
    - 如果下中天仍然在地上 -> 永不下降的天体
+ 太阳的位置：赤道坐标系下（赤经和赤纬描述，与局域地点和时间无关）

### 周年视运动
#### 太阳的周年视运动
+ 回归年： 太阳的黄经从春分点起算，沿逆时针度量。太阳连续两次经过春分点的时间间隔为一个回归年。
+ 二十四节气时间间隔不严格均匀：由于地球的公转轨道是椭圆，轨道运动速度不均匀（开普勒第二定律），故按照太阳的黄经值等分的二十四节气的时间间隔不均匀；  
+ 任一节气到下一年同一节气的时间间隔是严格相等的，为一个回归年

#### 地球的复杂运动带来的现象
+ **地球进动**：自转轴沿着另外一个轴转动。
    - 效果：黄赤交角不变，引起赤道面运动，春分点位置变化。
    - 回归年：太阳两次经过春分点（远处恒星背景下相对固定的点）的时间间隔
    - 恒星年：太阳两次经过同一恒星背景的时间间隔
    - 岁差：地轴进动时， 赤道面与黄道面的交 线也以同一周期在黄道面上移动。进动周期为26000年，对应每年春分点西移50角秒。  
+ **地球章动**：自转轴来回振动
    - 效果：黄赤交角增大减小增大减小，引起赤道面运动，春分点位置可能变化。
+ **地球极移**：地轴的瞬时转动轴可能在地球内部发生变化，地理经纬度变化。
    - 效果：地球瞬时自转轴在地球本体内的运动。  
    - 影响：需要取平均值，得到国际协议原点
+ 历元更换：因为上述复杂运动因素，恒星的位置需要标明与之相对应的年代（历元），一般五十年更换一次。

#### 月相与月食
+ 月食一定发生在满月农历十五的时候，因为其他时间地球挡不住月球
+ 月亮渐渐变满的过程中称为上弦月，渐渐变缺的过程中称为下弦月
+ 月球的潮汐锁定：月球的公转周期和地球的自转周期相同
+ 月球上看地球位置不变，地球没有周日视运动
+ 地球上观测者只能看到月亮的一面。

### 时间与历法
#### 关于时间本身
+ 时间的概念：间隔和时刻
+ 时间计量的基本原则：用于表征时间的具体事物的运动规律已经掌握，选取其某一过程为时间的基本单位，选取该事务的某一状态的时间点作为计量的起算点。

#### 恒星时、真太阳时、平太阳时
+ **恒星时**：
    - 时间单位：**春分点**（黄道面和赤道面的交点，如果不考虑地球的进动，春分点在恒星背景下是一个相对固定的点）连续两次**上中天**的时间间隔
    - 起始点：春分点上中天
    - 恒星时的数值上等于春分点的时角
+ **真太阳时**：
    - 时间单位：**真太阳**连续两次**下中天**的时间间隔
    - 起始点：下中天
    - 真太阳时的数值等于12小时加上太阳的时角

<img src="https://cdn.nlark.com/yuque/0/2025/png/2371504/1758640348201-2750f606-c6b7-43c1-80ba-dfabbade8475.png" width="398" title="" crop="0,0,1,1" id="t2KCj" class="ne-image">

    - 真太阳时与恒星日的差异：真太阳日比恒星日长大约4分钟，因为地球绕着太阳逆时针公转，而地球自转也是逆时针。
    - 真太阳时的缺陷：
        * 太阳相对地球的轨道为椭圆，运动速度变化。
        * 即使太阳在黄道上运动均匀，由于黄赤交角的存在，投影在赤道上的太阳时角变化也不均匀。  
        * 由此可知，真太阳时不是均匀的时间计量系统
+ **平太阳时**：两个辅助点，最后一个辅助点是在赤道上，以太阳平均速度运动的平太阳。
    - **第一辅助点——黄道平太阳**。在黄道上做匀速运动，其视运动的速度等于太阳时运动速度的 平均值，并且与真太阳同时经过近地点和远地点。  
        * 效果：让太阳在黄道上匀速运动
    - **第二辅助点——赤道平太阳**。在赤道上做匀速 运动，不仅与第一辅助点速度相同，而且两者同时经过春分点和秋分点。
        * 效果：让太阳转换到赤道上匀速运动。
        * 这么做的原因：时角的规定是基于赤道的。
        * 
+ 三者的共同特点：恒星时、真太阳时、平太阳时是以春分点、真太阳、平太阳为参考点，以过当地子午圈的时刻为起算点，以时角度量的。**对于观测者，只要位于不同的地理经圈，就对应不同的天子午圈，因此，参考点经过的天子午圈不同，所得时刻也不同。**
+ **注意事项**
    - 恒星时没有日期的概念，通常所说的恒星时某时，日期是平时系统的日期。
        * 只要给出了他们的平时时刻，就能判断先后；如果知道恒星时时刻，需要知道顺序。
    - 恒星时零点：春分点上中天。
        * 春分日：春分点上中天，恒星时0时；太阳上中天，太阳时12时。
        * 秋分日：秋分点上中天春分点下中天，恒星时12时；太阳时12时。

#### 地方时、世界时、区时
+ 地方时：以**本地子午面**为起算平面，根据任意量时天体所确定的时间。
+ 世界时：地理经度λ＝0º的地方平太阳时M
    - 世界时的几个版本：天文台直接测量得结果，经度修正，季节修正
+ 区时：把全球分成24个时区，每区跨经度15º，各区把中央经线的地方时作为本区统一使用的标准时。这样的区域称为时区 ；这样的时间称为区时。

#### 历法
+ 历法：推算年、月、日的时间长度，协调它们的关系，制定一定的时间序列法则。
    - 基本单位：年（回归年）、月（朔望月）、日（平太阳日）
+ 历法的制定原则
    - 反映天文客观规律，正确的反映天象和四季变化
    - 日历要简单明了容易记忆
    - 有通用性
    - 需要规定：纪元——开始计算的年代；岁首——规定一年的开端。规定每年所含的日数，月份的划分，每月所含天数。
+ 一些例子：
    - 太阴历：（回历）以朔望月为基本单位。
    - 太阳历：（公历）以回归年为基本单位。
    - 阴阳历：（农历）以朔望月计算月，以回归年计算年，二者兼顾。
    - 儒略历：一种不用年、月的长期纪日法。
