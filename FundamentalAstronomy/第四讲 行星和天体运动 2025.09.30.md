### 太阳系概况
#### 太阳系成员
+ 太阳：质量为太阳系质量的99.87%
+ 行星：水星，金星，地球，火星，木星，土星，海王星
+ 矮行星：冥王星，谷神星，阋神星等。

#### 行星的视运动
+ **地内行星的运动**
    - 合：行星和太阳的黄经相同时，称为行星的合日，简称“合”
    - 大距：行星与太阳角距离达到最大值时，称为“大距”
    - 凌日：地内行星下合
    - 地内行星在一个回合周期中的过程：

上合 - 顺行 - 东大距 - 顺行 - 留 - 逆行 - 下合 - 逆行 - 留 - 顺行 - 西大距 顺行 - 上合

    - **先东大距再留的原因**：从地球角度来看，太阳绕着地球转，金星绕着太阳转。东大距时刻，金星速度直指地球，垂直于视线方向分量为零，出现东大距，进行随着太阳继续顺行一段距离，相对太阳垂直视线方向分量与太阳运动方向相反，逐渐留，再逆行。

+ **地外行星的运动**
    - 合：行星的太阳的黄经相同。
    - 冲：行星和太阳的黄经相差180度。
    - 东方照、西方照：行星和太阳的黄经相差90度。
    - 视运动过程：

合→（顺行）→ 西方照→（顺行）→ 留→（逆行）→ 冲→（逆行）→ 留→（顺行）→ 东方照→（顺行）→ 合

+ 回合周期：地球上连续观测到两次上合或者冲的时间间隔。

#### 水星（Mercury）
+ 轨道：公转轨道是椭圆
+ 亮度：平均视星等-1.9等
+ 大气：无法留住稳定的大气层
    - 仅有一个稀薄的外逸层（exosphere，原子不断流失和补充）
    - 在水星表面赤道区域，白昼的温度高达+430℃，而在夜间的温度降至－170 ℃左右。

#### 金星（Venus）
+ 亮度：最亮时视星等-4.6等，是夜空中仅次于月亮第二亮的天体
    - 中国古代被称为太白金星，日出前出现在东方称启明，日落后出现在西方时被称为长庚。和太阳最大的角距离最大为47.8度。

+ 大气：浓密的大气，高温且具有腐蚀性
    - 温度高达 450℃以上，气压高出地球的近百倍。大气中还频繁的出现闪电现象。 
    - 金星的温度一般都在465℃～485℃之间，有90个大气压，这相当于地球的海底深900米处的压力。
    - 金星的云层主要集中在40-65公里高度，33公里以下的大气就清澈了。
    - 金星大气中二氧化碳居多，达到97%以上，还有近3%的氮和很少量的水蒸气、一氧化碳等。 
    - 金星高云层中风速可达400km/s，是金星自转的60倍。
+ 没有磁场。

#### 火星（Mars）
+ 别名：古代称为“荧惑”或者“惑星”
+ 亮度：最亮时视星等为-2.01等，最暗时为1.5等。
+ 大气：
    - 远比地球大气稀薄，气压仅为地球大气压的0.5％ ～0.8％。
    - 其主要成份是二氧化碳，占95％。
    - 氮占3％，水汽仅占0.01％。
    - 火星云层的主要成分是干冰。
+ 没有磁场，太阳风对火星大气层不断侵蚀
+ 卫星：两颗卫星，形状不规则，发生潮汐锁定
    - 火卫一：Phobos，较大，西升东落
    - 火卫二：Deimos，较小，东升西落

#### 小行星带和谷神星
+ 小行星的共性
    - 形状：多数形状不规则，少部分是球形
    - 反照率：小行星的表面的反照率各不相同，说明它们的组成物质是不同的。反光能力大的是石质小行星，反光能力差的是碳质小行星。
    - 卫星：有些小行星甚至还有卫星
    - 分布位置：大部分位于火星和木星轨道中间的小行星带，少数到处运动。小行星群包含特洛伊群和希腊群。
+ 近地小行星分类
    - 阿坦型 （Aten） a＜1AU 远日距略大于1.0AU， 
    - 阿波罗型（Apollo） a：1.08—4.2AU 近日距＜1AU
    - 阿莫尔型（Amor） a＞1AU， 近日距≥1AU

+ 谷神星（Ceres）：
    - 内太阳系唯一的矮行星，小行星带内最大的天体。
    - 半径476km，距太阳2.8AU，0.00015M⊕
    - 2006年被国际天文学联合会列为**矮行星**：
        * 位于围绕太阳的轨道上
        * 有足够大的质量克服固体应力达到流体静力学平衡的形状（接近球体）
        * 未能清除轨道附近的小天体
        * 不属于卫星。

#### 木星（Jupiter）
+ 别名：公转周期约为12年，古代称为“岁星”
+ 亮度：最亮时视星等位-2.94等，最暗时视星等为1.6等
+ 大气：
    - 自转速度最快，赤道旋转周期为9h50m，高纬度地区旋转周期为9h55m
    - 赤道对轨道面夹角为3度，没有明显季节变化
    - 浓厚的大气，随着大气层向下温度和压力增加，氢变为液态（没有明显汽、液界线的流体状态）
+ 磁场：木星有很强的磁场
+ 卫星：木卫一（Io），木卫二（Europa），木卫三（Ganymede），木卫四（Calisto）

#### 土星（Saturn）
+ 别名：中国古代也称为“镇星”。
+ 亮度：平均视星等0.67等，用小望远镜就可以观测到明亮的土星环。
+ 大气：
    - 自转速度最快，赤道旋转周期为9h50m，高纬度地区旋转周期为9h55m
    - 赤道对轨道面夹角为3度，没有明显季节变化
    - 浓厚的大气，随着大气层向下温度和压力增加，氢变为液态（没有明显汽、液界线的流体状态）
+ 磁场：土星同样存在磁场，强度大约是木星的1/12，比地球磁场略弱。
+ 卫星：土星有274个卫星，最大的为土卫六(Titan)，直径5150km。土星卫星中最大的一个，也是太阳系第二大卫星。
+ 土星环：由无数细小颗粒组成（μm~m），主要成分是水冰。尺度延展达几十万公里，厚度约10米。

#### 天王星（Uranus）
+ 亮度：视星等约6.5等。
+ 大气：
    - 天王星的自转很特殊，自转周期是17.2小时，自转轴几乎就在公转的轨道面上，行星的赤道面与轨道面的交角约98°
    - 而且它自转方向和金星一样，也是由东向西的。
    - 天王星的表面上包围着很厚的大气层，地面观测难以看清。
    - 由行星探测器探测得知天王星的大气的主要成分是由氢和氦组成。
    - 内部主要由冰构成，包括水，甲烷，氨等成分。
+ 磁场：存在磁场，磁轴与自转轴夹角约60度。
+ 卫星：土星有274个卫星，最大的为土卫六(Titan)，直径5150km。土星卫星中最大的一个，也是太阳系第二大卫星。
+ 土星环：由无数细小颗粒组成（μm~m），主要成分是水冰。尺度延展达几十万公里，厚度约10米。

#### 海王星（Neptune）
+ 亮度：海王星离太阳比较远，看起来比较暗弱，亮度约 7.8等。
+ 大气：
    - 赤道的自转周期为16.5小时，两极的自转周期为14.2小时。
    - 赤道与轨道面交角约28.3°，与地球差不多，因此海王星上也应有四季变化。每个季节长达41个地球年。
    - 较高的反射率，表明它有浓密的大气层。大气的主要成分有甲烷和氢。 
+ 有16颗卫星（海卫一Triton最大，半径1353km），有5道光环，浓厚的大气，有磁场

    - 陨石和陨铁，按照不同化学成分进行区分。
+ 柯伊伯带，奥尔特云。
    - 柯伊伯带：30~55AU，doughnut-shaped ring，短周期彗星，小行星，矮行星。
    - 其外部更远的区域为离散盘，其中天体轨道部分在柯伊伯带内，但是延展到更远的区域。
    - 奥尔特云，~10^4 AU的球层区域，长周期彗星的发源地（轨道周期超过200年）。
