### 双星的类型
#### 双星系统
+ 双星系统：两颗星绕共同质心旋转的系统，两颗星叫系统的子星
    - 较亮的星叫主星，较暗的星叫伴星。
    - 注意是按照亮度来分的，而不是质量。

#### 双星的类型
+ **目视双星**：用眼睛或望远镜内能分辨出的两颗子星的双星系统。
    - 大部分都不能用眼睛分辨。
    - 一般来说，目视双星互相绕转周期很长（距离比较远），数年至数百年不等。

<img src="https://cdn.nlark.com/yuque/0/2025/png/2371504/1762260455775-adb5c045-3b24-43a3-b42d-3a106e440cef.png" width="444" title="" crop="0,0,1,1" id="u137aedf9" class="ne-image">

    - 可以通过测量周期和相对轨道半长轴计算出总质量：

$ \frac{a^3}{T^2} = \frac{G M_总}{4\pi^2} $

        * 或者可以写成分着的形式：

$ \frac{(a_1+a_2)^3}{T^2} = \frac{G (M_1+M_2)}{4\pi^2} $

    - 如果子星较暗
        * 如果没有自行，主星看上去在原地旋转；伴星也在原地旋转（只是看不见）。
        * 如果有自行，也就是公共质心在自行。看上去主星的轨迹如波浪形运动；当然伴星的运动轨迹也是波浪形的，如天狼星。

<img src="https://cdn.nlark.com/yuque/0/2025/png/2371504/1762260810872-28417bfa-a7c2-43c2-b683-df158b22464c.png" width="291" title="恒星的自行" crop="0,0,1,1" id="u57ee467d" class="ne-image">

+ 自行：
    - 恒星在天球上的视运动有两种成分：
        * 地球和太阳的运动引起的相对运动
        * 恒星的真实视运动
    - 恒星的真实运动速度可以分解为两个分量：
        * 横向速度（自行）
        * 视向（或径向）速度
+ **分光双星**：通常靠的太近，用望远镜无法直接分辨
    - 通过子星轨道运动引起的谱线**多普勒位移**确定双星性质

+ 质量函数：可以给出不可见半星的质量下限。【计算比较复杂】
    - 实际上就是用已知参数去限制。
+ **食变双星**：子星相互交食造成的亮度变化的双星。
    - 观测者视线与双星运动轨道面接近，往往意味着两个子星相距很近
    - 光变曲线：子星间的相互交食造成的双星亮度的变化曲线。

#### 洛希瓣和双星分类
+ 1864年,法国数学家洛希(E. Roche)首先提出计算密近双星系统引力势简化模型
    - 假设1：每个子星的内部密度分布是球对称的，在动力学上可以认为是质点
    - 假设2：两子星以圆轨道绕公共质心运动
    - 假设3：两子星的自转和公转一致
+ 洛希瓣与拉格朗日点
    - 内临界等势面:同时包络两颗子星并且相接于其间一点(L1)的等势面；在L1(内拉格朗日点),两颗子星对物质产生的作用力正好相等, 洛希势达极大值
    - 外临界等势面：另外一等势面，过外拉格朗日点L2（很多人造卫星到达点）；物质可从此处溢出
    - 洛希瓣:由临界等势面包围的空间

<img src="https://cdn.nlark.com/yuque/0/2025/png/2371504/1762758097604-95f38e61-c048-46c7-9a71-9c474db5fcb6.png" width="256" title="内临界等势面" crop="0,0,1,1" id="uea41b1f7" class="ne-image">

+ 双星的分类：根据双星中的一颗或两颗子星是否充满洛希瓣
    - 不相接双星: 两颗子星均未充满洛希瓣
    - 半相接双星:一颗子星充满洛希瓣，如天琴β
    - 相接双星:两颗子星均充满洛希瓣，有共同对流包层，如大熊W

+ 双星中的物质传输
    - 星风传输（不相接、半相接双星）：大质量恒星在演化过程中通常有强烈的星风
    - 洛希瓣渗溢 （半相接、相接双星）：形成吸积盘
+ 大陵五佯谬：
    - 为什么质量小的反而演化的快？
    - 双星系统应该是相同时间产生的。
    - 亚巨星子星的前身星是此双星系统中质量较大的主序星，它先演化充满洛希瓣，物质传输使得两子星的质量发生逆转。
    - 吸收质量的恒星英文形象地翻译为vampire star

### 变星
+ 变星分为三类：脉动变星、爆发变星、食变星
+ 物理变星：恒星内部发生变化，占80%。
    - 脉动变星：占物理变星的90％，光变由星体脉动引起
    - 爆发变星：光变由一次或多次的周期性爆发引起
+ 几何变星：位置关系变化使观察到的图像发生变化，占20%

#### 脉动变星
+ 脉动变星：脉动变星: 星体发生有节律的、大规模运动而使亮度发生变化。

$ R ↑→ T ↓ → R ↓ → T ↑ → R ↑ $

+ 造父变星 (Cepheid variables) 
+ 周期性脉动变星 P～1-50 d 
+ 原型:造父一(仙王δ，1784年被发现光变) P = 5.4 d

<img src="https://cdn.nlark.com/yuque/0/2025/png/2371504/1762859120984-d65c9608-5ab3-4f90-80d5-07e6e7b83e7e.png" width="398" title="脉动变星的分类" crop="0,0,1,1" id="u1448606b" class="ne-image">

+ 调控机制：
    - 温度改变，氦元素的电离比例改变（一次电离还是二次电离）
    - 如果温度下降，二次电离的减少，一次电离的氦离子电子可以吸收光，不透明度增加，吸收热量增多，温度上升。周而复始。
    - 大质量造父变星脉动周期长。 -> **周光关系**
+ 根据周光关系，测量周期可以得到实际光度，对比接收到的光度，也就可以得到距离大小。

#### 爆发变星
+ 爆发变星：包括**超新星**和**激变变星**。
+ 激变变星：包含一颗白矮星的半相接双星
    - 分类：新星 (novae) ，再发新星，类新星变星，矮新星，磁激变变星
    - 再发新星的激波过程：后一次喷发的追上了前一次的，形成了团块。

### 星团
+ 恒星在天空中的分布有聚集成团的现象。
+ 根据它们在星系内的空间分布、运动性质及其一些物理特性明显地不同可以分为不同类型的恒星集团。
+ 大致可分为两种：**疏散星团**与**球状星团**。

#### 疏散星团
+ 在银河系中已发现1000多个疏散星团（Open Clusters）。
+ 每个疏散星团包含几百到几万颗恒星
+ 尺度在几个pc。
+ 成员星比较年轻，也有中等年龄的恒星。

+ 在银河系中已发现150多个球状星团（Globular Clusters）。和疏散星团相比

    - 分布更致密
    - 年老恒星

### 天体测距
+ 三角视差（trigonometric parallax）
    - 周年视差（annual parallax）：以地球公转轨道半径作为基线测量恒星的距离周年视差p是恒星相对于地球轨道半径（1AU）所张的夹角
+ 分光视差（spectroscopic parallax）
    - 对于主序恒星，可以根据观测得到的恒星的色指数（光谱型），利用赫罗图估算其光度，再结合视星等估算其距离；
    - 对于星团，可以认为其成员星到我们距离相同，如果其中有大量主序星，可以通过移动星团主序与赫罗图主序重合来估算整个星团的距离。（大量数据拟合）
    - 典型：造父变星
