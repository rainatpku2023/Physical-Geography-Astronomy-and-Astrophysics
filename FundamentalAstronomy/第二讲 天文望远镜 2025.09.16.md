### 天文望远镜与探测器
+ 一些探测器件
    - 照相底片：模拟信号
    - 光电倍增管
    - CCD（电荷耦合器件）：灵敏，线性，低噪声，数字化
    - CMOS：未来一大方向
+ 使用望远镜的目的
    - 看得见：增加接受接收光子的总和，提高灵敏度
    - 看得清：提高分辨率
        * 空间分辨率 -> 依赖于光学系统
        * 频谱分辨率 -> 依赖于终端仪器
        * 时间分辨率 -> 依赖于终端仪器
+ 望远镜的性能指标
    - 聚光能力：也就是有效口径，天体成像亮度正比于有效镜面面积
    - 角分辨率：取决于光的衍射，埃里斑
        * 1弧度 = 3438角分 = 206265角秒
        * 理论上眼睛太阳光分辨率23角秒，1米望远镜0.14角秒（视宁度影响，基本无法达到）
        * 视宁度：<font style="color:rgb(17, 17, 17);">用于描述天文观测的目标受大气湍流的影响而看起来变得模糊和闪烁程度的物理量</font>

### 光学系统组成与问题
+ 镜子分类
    - 物镜：收集天体来的辐射并聚焦出天体像，以便观测研究。
        * 物镜有折射镜、反射镜、透镜与反射镜的组合。
    - 目镜：观看物镜焦平面上的天体像。
    - 主镜：视场小，便于观测
    - 寻星镜：视场大，便于寻星；现代望远镜的指向精度非常高
    - 导星镜：用于跟踪星体
+ 放大倍数
    - 焦比：焦距比上口径
    - 放大倍数：角放大率等于物镜焦距与目镜焦距之比
        * 目镜焦距越短可以获得更大的放大率，但是这样单位时间内进入镜筒的光子数减小，会让观测挑剔变得非常暗和模糊[Telescope Simulator](http://astro.unl.edu/classaction/animations/telescopes/telescope10.html)
    - 思考：同一个望远镜先看月球再看土星，不需要二次调焦，因为月球和土星都太远了，可以视为平行光。
    - 注意：口径和放大倍数都很重要
+ CCD图像：
    - 用底片比例尺衡量望远镜性能，定义为底片（CCD）中央每1mm对应的星空焦距
+ 光路分类：折射望远镜、反射望远镜、折反式望远镜
+ 望远镜的问题
    - 色差：不同颜色光的焦距不一样
    - 球差：离光轴较近的光焦距更远
    - 彗差：与光轴倾斜的光在焦面的像非点源
    - 大气干扰

### 望远镜按照光路结构分类
#### 折射望远镜
+ 折射望远镜分类
    - 伽利略式：凸透镜物镜，凹透镜目镜；正像；视场小、放大率小 -> 市场售卖
    - 开普勒式：凸透镜物镜，凸透镜目镜；正像；视场大、放大率大 -> 天文观测
+ 优点：焦距较大、相对口径较小，工作视场大；对镜筒弯曲不敏感
+ 缺点：有残余色差，对紫外、红外波段的辐射吸收厉害，难以浇制和磨制

#### 反射望远镜
+ 主轴式、牛顿式、卡塞格林式、折轴式

<img src="https://cdn.nlark.com/yuque/0/2025/png/2371504/1758024017115-0932bc13-8b9c-4ee4-a7a6-33593cb76434.png" width="397" title="" crop="0,0,1,1" id="u6040e11a" class="ne-image">

+ 优点：口径大，无色差，对镜面材料要求低，易磨制
+ 缺点：视场较小，镜面互相挡光。需重新反复镀反射面，部件组装、校准困难
+ 随着技术的进步，上述缺点大多已经克服。所以现代大型望远镜大都是反射望远镜，尤其是R-C系统。

#### 折反式望远镜
+ 优点：视场大、成像质量好（改正镜改正了像差）。适用于观测有视面天体（行星、星云）。适于巡天和流星、人造卫星观测
+ 缺点：改正镜难磨制，不能做得很大（随着技术的进步，部分可以克服）

### 机械装置
+ 包括基座及上面两个相互垂直的**转轴**、**刻度盘**及**指标**
    - 便于望远镜灵活指向天体。
    - 按转轴方向常采用地平式和赤道式两类。
+ 地平式装置
    - 两个轴分布在垂直和水平方向。
    - 绕垂直方向转动可对向天体的地平经度（方位角），绕水平轴转动可对向天体的地平纬度。特点：稳定性好，重量低。
+ 赤道式装置
    - 两个轴是极轴和赤纬轴。极轴平行于地球自转轴，而赤纬轴平行于赤道面。
    - 绕赤纬轴转动可对向天体的赤纬
    - 绕极轴转动可对向天体的时角且易跟踪天体的周日视运动

### 空间天文学与空间望远镜
+ 由于大气窗口，很多波段只能在太空观测
+ 不受大气影响，图像质量可直接达衍射极限
+ 极低的背景天光，大大提升探测灵敏度
+ 不受环境影响，如地面灯光
+ 不受天气影响，如刮风下雨
