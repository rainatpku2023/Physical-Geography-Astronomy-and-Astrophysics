### 恒星的基本物理量及其测量
#### 恒星的光度
+ 光度（Luminosity）：天体在单位时间内辐射的总能量，是恒星的固有属性。
    - 恒星的光度由三部分组成：光子携带的光度、中微子携带的光度、质量抛射带走的质量。
    - 恒星光度的范围：$ 10^{-5}\sim 10^6 L_☉ $。高光度的恒星很稀少，大部分光度还不如太阳。
+ 光度函数（Luminosity Function of the Nearby Stars）：不同绝对星等下恒星个数分布。

#### 恒星表面温度
+ **有效温度**：描述恒星光球辐射的普朗克公式对应的温度，即相同光度的黑体的温度。
+ **色温度**：具有不同温度的黑体，辐射强度随波长/频率的分布不同，观测上表现为颜色的差异。
    - 由维恩位移定律，恒星的温度越高，最高发射率的温度越来越高，恒星越来越偏向紫色。
    - 据此，可以用普朗克公式给出的功率-辐射分布曲线来拟合。
    - 一般定义的色温度都略高于有效温度，特别当恒星表面温度非常高时。
+ **色指数**（Color index）：在不同波段测量得到的星等之差,如U-B, B-V等。
    - 𝐵 − 𝑉 = 𝑚𝐵 − 𝑚𝑉 = 𝑀𝐵 − 𝑀𝑉
    - 𝑈 − 𝐵 = 𝑚𝑈 − 𝑚𝐵 = 𝑀𝑈 − 𝑀𝐵
    - **<font style="color:rgb(15, 17, 21);">U波段</font>**<font style="color:rgb(15, 17, 21);">：</font>**<font style="color:rgb(15, 17, 21);">紫外</font>**<font style="color:rgb(15, 17, 21);">波段，中心波长约365纳米。</font>
    - **<font style="color:rgb(15, 17, 21);">B波段</font>**<font style="color:rgb(15, 17, 21);">：</font>**<font style="color:rgb(15, 17, 21);">蓝色</font>**<font style="color:rgb(15, 17, 21);">波段，中心波长约445纳米。</font>
    - **<font style="color:rgb(15, 17, 21);">V波段</font>**<font style="color:rgb(15, 17, 21);">：</font>**<font style="color:rgb(15, 17, 21);">视觉</font>**<font style="color:rgb(15, 17, 21);">波段，中心波长约550纳米。</font>
    - 色指数的计算会受到星际消光的影响。
    - 星际消光影响因素：天体的光度，天体的距离，星际物质对于辐射的吸收和散射。
    - **色余**：由于星际消光（红化）我们观测到的色指数与真实色指数（未经过星际介质吸收的）之间的差异。

$ Color excess_{B-V} =E_{B-V} =(B_{obs}-V_{obs})-(B-V) $

+ **热星等**（bolometric magnitude）：恒星的辐射对全波段积分对应的星等为热星等
    - **热改正**：热星等与目视星等之间的改正值称为热改正（BC）

$ m_{bol}=m_V+BC，M_{bol}=M_V+BC $

+ Harvard光谱分类：**光谱型**（Spectral type）
    - Oh Be A Fine Girl Kiss Me，从左到右温度越来越低。
    - 温度越来越低，被电离的物质越来越少，能级结构越来越复杂，吸收线越来越多。
    - A型星的作用：明显的氢吸收线，可以用于光谱定标。

#### 恒星的质量
+ **恒星质量的测量**：双星的质量和可以按照开普勒第三定律计算出来。
+ **质量范围**：$ 0.1M_⊙\sim120 M_⊙ $
    - 60倍以上太阳质量的恒星是动力学不稳定的。
    - 0.08倍太阳质量以下的恒星无法点燃氢燃烧，如褐矮星。
+ **密度范围**：超巨星10-6 g cm-3；太阳1.4 g cm-3；白矮星106 g cm-3
+ **质光关系**：恒星的光度与质量的某次方成正比
    - 主序星，光度大概与质量的3.5-4次方成正比。因此寿命和质量-2到-1.5次方成正比。
    - 质量大的恒星寿命短。

$ L/L_⊙=(M/M_⊙)^{4.0±0.02} \; \mathrm{for} \; 0.4M_⊙<M<10M_⊙ $

$ L/L_⊙=(M/M_⊙)^{3.6±0.1} \; \mathrm{for}\;  5M_⊙≤M≤40M_⊙ $

#### 恒星的半径
+ **恒星半径的测量**：
    - 比较近的恒星：干涉法(Interferometric measurements)和月掩星法(Lunar occultations)
        * 掩食双星(Eclipsing binaries),可利用光变曲线的形状和掩食的持续时间来测定

<img src="https://cdn.nlark.com/yuque/0/2025/png/2371504/1761910865221-0010c6d1-3a4f-40f9-9777-3bb2e7ecf7c3.png" width="520" title="掩食双星光变曲线示意图" crop="0,0,1,1" id="u6676c296" class="ne-image">

    - 比较远的恒星：辐射方法测量
+ **恒星按照体积分类**：
    - 超巨星（supergiants）$ R \sim100-1000 R_⊙ $
    - 巨星（giants） $ R \sim 10-100 R_⊙ $
    - 矮星（dwarfs） $ R\sim R_⊙ $或$ R\sim R_⊙ $
+ **恒星半径范围**：从小于0.01倍太阳半径到大于100倍太阳半径
    - 太阳半径：$ (6.9598±0.0007)×10^{10}\mathrm{cm} $

+ 主序星的质量-半径关系：半径正比于质量的0.5到1次方。半径越大的恒星密度越小。

### 赫罗图
#### 赫罗图
+ 按照**恒星的光度**$ L $（或者用绝对星等$ M $衡量）和**恒星的有效温度**（或者用色指数$ C $， 光谱型）
    - 前者描述了总的辐射功率，后者描述了功率按照频率分布。
+ 丹麦天文学家$ \mathrm{E. Hertzsprung} $和美国天文学家$ \mathrm{H.R. Russell} $绘制的**光度-温度分布图**。
    - 横轴：温度，或者OBAFGKM。
    - 纵轴：光度或者绝对星等。

#### 主序
+ 主序（Main sequence）：大量恒星在赫罗图上分布在从左上到右下的带状区域，称为主序
    - 高质量的恒星明亮且温度高，位于主序带的上部。
    - 低质量的恒星黯淡且温度低，位于主序带的下部。
    - 主序星满足温度越高光度越高的规律。

+ 恒星在赫罗图上的运动：
    - **注意恒星的质量不会增大，所以不会沿着主序星直线走很远。**恒星走一段就脱离主序了。
    - **质量越大的恒星寿命越短，越早脱离主序**。
    - 脱离主序后的演化历程由初始质量决定。
    - 虽然蓝巨星质量大总功率大，但因为因为其温度高，在主序内。
    - 红巨星质量大功率大，但温度不高，所以在主序以外。小质量的恒星常常演化为红巨星。红巨星是老年形态。

#### 恒星的光度级分类
+ Yerkes光谱分类根据谱线宽度的变化,对恒星进行光度分类。 以罗马数字表示，数字值越高，则谱线越宽。
    - 分子频繁碰撞，相互作用明显，能级被扰动，谱线展宽明显。

### 恒星的演化

#### 恒星的形成
+ 恒星形成的一般过程：
    - 含气体和尘埃的分子云在自引力作用下塌缩
    - 原恒星(protostar)
    - 成为主序恒星

<img src="https://cdn.nlark.com/yuque/0/2025/png/2371504/1761913240415-de886ed5-f9ad-40ce-8e92-bd8614becdfb.png" width="483" title="原恒星系统" crop="0,0,1,1" id="ud013de5c" class="ne-image"><img src="https://cdn.nlark.com/yuque/0/2025/png/2371504/1761913217671-0ffea844-14c5-4761-9e61-bf334d05fee7.png" width="145" title="原恒星风" crop="0,0,1,1" id="u3f0c0296" class="ne-image">

+ Hayashi track：原恒星演化在这一过程中有剧烈的表面活动，产生强烈的原恒星风，以金牛座发现的原恒星T Tauri为代表，因此也称为T Tauri phase。

#### 恒星的演化
+ 零龄主序：
    - 刚刚开始核心H燃烧的恒星,在H-R图上占据主序带的最左侧。
    - 核反应4 1H → 4He；核心区温度Tc↑,核反应产能率ε↑→ 光度L↑→恒星半径R↑→ Te ↓

<img src="https://cdn.nlark.com/yuque/0/2025/png/2371504/1762258234286-399a1f3c-dbe7-4a3d-83ca-41bed1842eea.png" width="306" title="主序带：主序星从核心H燃烧开始到结束在H-R图 上占据的带状区域。" crop="0,0,1,1" id="u21531f5b" class="ne-image">



+ 典型恒星的演化通常要经历: 
    - 核心**氢燃烧**的**主序星**阶段（Main Sequence）
    - 核心**氢燃烧枯竭**后的**红巨星**阶段（Red Giant Branch）
    - 进入核心**氦燃烧**的**水平支**阶段（Horizontal Branch）
    - 核心**氦燃烧枯竭**后的**渐进巨星支**阶段（Asymptotic Giant Branch, AGB）
+ 水平支：一个新的“主序”？
+ 碳星：白矮星(CO核心坍缩成白矮星)：太阳质量的白矮星，半径和地球相当，电子简并压主导。
    - 一块大钻石，但是电子简并压主导，比地球上的钻石硬很多。
+ 演化的结局：
    - 小质量恒星形成**行星状星云(PN)和白矮星(WD)**
    - 大质量恒星经过**超新星爆发形成中子星或黑洞**
