### 太阳系天体的运动规律
+ 行星轨道要素（6个自由度需要6个约束条件消除）：
    - 椭圆轨道半长轴：限制轨道尺寸
    - 轨道偏心率：限制轨道形状
    - 轨道倾角：行星轨道面与黄道面夹角 -> 限制章动角
    - 升交点黄经：升交点到春分点的角距 -> 限制进动角
    - 近日点角距： 升交点到近日点的角距-> 限制自转角
    - 近日点时刻： -> 确定卫星在轨道上位置与时间的对应关系
+ 行星和卫星的轨道特征
    - 共面性；近圆性；同向性
+ 轨道半径经验规律：提丢斯-波德定则

+ 开普勒三定律：椭圆定律、面积定律、调和定律
    - 椭圆定律：行星运动的轨道是椭圆，太阳位于椭圆的一个焦点上
    - 面积定律：行星绕太阳运动时，以太阳为坐标原点的行星向径在相等的时间所扫过的面积相等
    - 调和定律：不同行星在轨道上公转周期 T 的平方与行星轨道半长径 a 的立方成正比

### 太阳系的起源和演化
#### 星云假说
+ 太阳系的诞生地称为原始太阳星云，原为一大团炽热的、缓慢旋转的气体尘埃星云。由于引力的作用，它逐渐收缩、凝聚，同时旋转速度加快，逐渐形成一个“铁饼”状的星云团块，这个引力团块的核心部分形成了太阳
+ 那些远离星云中心的旋转气体环不断收缩，逐渐变冷。这些环系在运动中不断受到其他团块的撞击，分裂为许多行星，并逐渐形成行星系统，最后形成了太阳、大行星和无数小天体所组成的太阳系家族

#### 星云假说理论的改进和完善
+ 行星的形成：红外辐射冷却，尘埃形成凝结核吸附周围物质
+ 太阳系物质组成差距：
    - 内太阳系，温度无法让水、甲烷等凝结，形成类地行星
    - 外太阳系，足够冷，允许水、氨、甲烷等凝聚成固体，可以俘获H2等，可以形成类木行星
+ 太空形成：年轻太阳的太阳风最后清空行星盘残余的气体和尘埃物质
+ 小行星带的形成：很可能是由于附近大行星的引力作用，使木星火星之间的小行星体最终没有能形成一个行星

### 太阳系外行星探索
#### 太阳系外行星的发现
+ **视向速度法（Radial Velocity）**：
    - 径向速度周期性变化
    - 恒星-行星系统绕其共同质量中心转动；恒星也运动，其辐射呈现规律的多普勒红移蓝移
        * -> 具体来说就是吸收暗线的红移和蓝移
    - 适用范围：行星质量较大，轨道面法向与视线方向夹角大
        * 即速度变化在径向分量大

<img src="https://cdn.nlark.com/yuque/0/2025/png/2371504/1760442488049-e6d1af99-d89d-4c91-82fa-608e8df6cc72.png" width="531" title="" crop="0,0,1,1" id="u1e54a634" class="ne-image">

+ **掩星法（Transit Method）**：
    - 行星绕恒星运动在，视线方向上对恒星的辐射造成遮挡，观测到的恒星光度（星等）降低
    - 恒星光度减弱的程度取决于被遮挡的面积（行星的半径）和恒星视圆面（恒星半径）的比例
+ **微引力透镜法（广义相对论效应）**
    - 恒星从背景恒星前穿过，对背景恒星产生微引力透镜效应，使背景恒星视亮度增亮。
    - 如果前景星有行星，则在光变曲线上会增加短时间迅速增亮的峰值。
    - 对大轨道半径的行星系统比较灵敏，对前面方法的有利补充。

+ **直接成像法**：如题，技术要求比较高
+ Astrometry: Minuscule Moments
    - 与RV同理，由于恒星-行星绕共同质心转动，恒星相对于周围恒星背景会有微小的摆动，通过高精度的天体测量，直接测到这个摆动，从而探测到系外行星系统。
    - 但是这种方法也有可能是伴星导致的，不一定是行星。

#### 宜居带行星 Habitable Zone Planets
+ 宜居带（Habitable Zone）
    - 恒星周围一个特定的距离范围。
    - 允许此间的行星表面存在液态水。

<img src="https://cdn.nlark.com/yuque/0/2025/png/2371504/1760443743259-54758111-1dad-49be-8f14-2ae0c4278bc6.png" width="500" title="" crop="0,0,1,1" id="u43191765" class="ne-image">

+ 系外行星分类
    - Gas Giant：类似于木星、土星L
        * 主要包含氢、氦
        * 没有固体表面
        * 质量甚至可以比木星大很多倍
    - Neptunian Planet：类海王星行星，类似于天王星、海王星
        * 有氢、氦大气
        * 以及核心，固态（冻结）表面等（水、甲烷，氨）
    - Super Earth
    - Terrestrial：类似于地球的固态行星
    - Hoptune：大小和海王星相近，类似于热木星，在距离中心恒星很近的轨道上被发现

### 天体的测光和分光观测
#### 一些物理概念：光度、视亮度、视星等、绝对星等
+ 测光：测量天体发光的强度
+ **光度（Luminosity，L）：天体在单位时间内辐射的总能量，即总的辐射功率**
    - 是天体的本征属性，不随观测者的差异而改变。
    - 单位：尔格，1erg/s = 10^-7 W
+ **视亮度（Brightness，F）：地球上单位面积接收到的天体辐射能量**
    - 与天体、接收者的位置有关
    - 单位：erg/s/cm2

$ L = 4\pi D^2 F $

    - 其中$ D $是观测者到恒星的距离，可见视亮度随着距离平方衰减。
+ **视星等（apparent magnitude，m）：**
    - 星等每降低5等，亮度变为原来的100倍

$ m_1-m_2 = -5\log_{100}({F_1/F_2}) = -2.5\lg (F_1/F_2) $

    - 采用定标常数表示：

$ m = -2.5\lg (F/F_0) $

        * 规定：

$ F = F_0 时m = 0 $

    - 视星等与天体、接受者位置有关
+ **绝对星等（absolute magnitude, M）：将天体放在10pc处的视星等**
    - 视星等与绝对星等的关系：

$ M-m = -2.5\lg (F_{10}/F_d) = 5-5\lg (d/1pc) $

    - 绝对星等与光度的关系：

$ M_1-M_2 =-2.5\lg (F_1/F_2) $

    - 太阳的绝对星等是：$ M_☉ = 4.75^m $
    - 太阳的光度是：$ L_⊙= 3.85×10^{33} \mathrm{erg\; s^{-1}} $

+ 总结：
    - 光度和视亮度是从功率角度定量描述天体发光情况的物理量。
    - 绝对星等和视星等是发光情况的天文化描述。
    - 光度和绝对星等之和目标恒星有关，与观测者到目标恒星的距离无关。
    - 视亮度和视星等和目标恒星、观测者到目标恒星的距离都有关。

#### 星等系统
+ 为什么有多种星等系统：星等值与零点选择有关

$ m = -2.5\lg (F/F_0) $

+ Vega系统：定义Vega（织女星）的所有波段星等都为零。问题是 Vega 有微弱的光变；而且随着测量精度的提高，零点需要不断的修正。
+ AB系统：使用流量来定义

$ m_{AB} = -2.5 \lg(f_ν) - 48.60 $

    - 其中 fν 为光的流量密度，单位为 erg s-1 cm-2 Hz-1; 注：1Jy(央斯基)=10-23 erg s-1 cm-2 Hz-1
    - AB星等和流量一一对应：
        *  fν: erg s-1 cm-2 Hz-1；fλ: erg s-1 cm-2 Å-1
        * νfν=λfλ; ν=c/λ; fν=(λ2/c)fλ

#### 测光系统
+ 例如Johnson系统，用光电倍增管和特定的紫、蓝、黄三种滤光片的光电光度计测光建立的‘UBV’系统。
    - 每一个波段算出来一个星等。
    - 之后还有人增加波段数，出现UBVRI系统。
+ 为什么要不同波段都测一次？因为不同恒星的发光情况不同。

#### 分光系统
+ 色散度：分解波长的能力，分为角色散度和线色散度
    - 角色散度：单位波长间隔的光被分解开的角度
    - 线色散度：单位波长间隔的光被色散的长度
+ 谱分辨本领：分辨最小波长差的能力
    - 棱镜摄谱仪谱分辨本领主要受准直镜口径D限制，R=(dθ/dλ)D
    - 光栅摄谱仪谱分辨本领主要受光栅总刻槽数N及光谱级m限制，R=Nm
    - 此外还需考虑光谱照度(与源亮度、望远镜口径有关)以及探测器效率等

#### 谱线波长的测定
+ 获得光谱后需要进行波长定标并证认各谱线波长。通常需用同一摄谱仪拍摄实验室光源的比较灯谱。光源常用铁、铁－氖、铁－氩－氖等阴极灯
+ 需要进行多普勒修正
