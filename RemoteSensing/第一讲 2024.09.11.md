### 遥感的概念
+ 遥感的概念：遥远的感知，即远距离不接触物体而获得其信息
+ 广义的遥感：在不直接接触的情况下，对目标物或自然现象远距离感知的一种探测技术
+ 狭义的遥感
    - 在高空和外层空间的各种平台上
    - 应用各种传感器获取地表信息 -> 不仅限于电磁波，可能还有重力场变化等
    - 通过数据传输和处理
    - 从而实现地面物体形状、大小、位置、性质及其环境的相互关系的一门现代化应用技术科学

+ 遥感的特点
    - 观测区域范围大，受条件限制少
    - 多尺度动态重复获取数据 -> 卫星的运行周期短，观测范围广
    - 电磁信息丰富、信息表达形式多样，获取信息的手段多、信息量大
    - 获取信息的速度快、周期短、重现性好 -> 有正式的处理程序
    - 数据可满足多用途监测、可比性强
    - 效益高、精度高、工作效率高 -> 只需要很小的工作团队

        * 光学遥感 Optical Remote Sensing
            + 研究对象的尺寸远大于光的波长，遵循几何光学的主要原理
            + 朗博定律：当两种介质交界面不光滑的时候（和光波长在一个数量级或者更大），可能发生漫反射
            + 高光谱数据：具有非常多个波段，可以将空间分别作为x、y轴，将波段信息作为z轴，得到一个三维数据立方体

<img src="https://cdn.nlark.com/yuque/0/2024/png/2371504/1726035630446-2cad47ae-e9e5-4d4b-bc93-0de8cc5a0229.png" width="393" title="" crop="0,0,1,1" id="uaf73b705" class="ne-image">

        * 热红外遥感 Thermal Infrared Remote Sensing
            + 探测波段在0.76-1000微米之间
            + 应用范围：地表温度反演、城市热岛效应、林火检测、灾害监测、探矿
            + 应用波段：3-5微米和8-14微米的大气窗口
        * 微波遥感 Microwave Remote Sensing
            + 探测波段在1mm-1m之间，一般分为毫米波、厘米波、分米波和米波
            + 利用微波设备来探测、接收被测物体在微波波段的电磁辐射和反射特性，识别远距离物体的技术
            + 特点 <- 微波波长长，波动性强，衍射效应显著；微波可由地球发出
                - 具有全天候昼夜工作能力，能穿透云层，不易收到气象条件日照水平的影响
                - 能穿透植被，具有探测地表下目标的能力
                - 获取的微波图像有明显的立体感，能提供可见光照相和红外遥感以外的信息
            + 主动探测：可以发射偏振程度高（Polarized）的微波，也可以进行调制（Modulation）
    - 按照成像信号能量来源
        * 被动遥感：分光辐射计、微波辐射计
        * 主动遥感：合成孔径雷达、真是孔径雷达、微波高度计、微波散射计、激光雷达
