### 植被的光谱特征
+ 植被的光谱特征
    - 可见光部分光合作用相关色素的强吸收
    - 近红外部分叶肉组织高反射，760nm红边（Red Edge）
        * 空腔的散射对于近红外波长范围内作用强
    - **近红外和短波红外部分水分吸收带**

#### 比例植被指数 RVI
+ 植被长得越好，红光反射率越低，近红外波段反射率越高
+ NIR-R平面：植被一直位于Soil-line直线的近红外高红光低的一侧
    - 为什么Soil-line是一条直线而不是一个点：土壤的反射率会随着有机质等成分变化而变化
+ 比值植被指数（Ratio Vegetation Index）
    - $ RVI = \frac{\rho_{NIR}}{\rho_R} $	
    - 范围广

#### 归一化植被指数 NDVI
+ $ NDVI = \frac{\rho_{NIR}-\rho_R}{\rho_{NIR}+\rho_R} $
+ 优势
    - 范围限制在0和1范围内。植被越多，NDVI越高
    - 数据容易获取，只需要两个波段
+ 遗留问题：
    - 不同土壤背景下相同生长状况NDVI也不同
    - 叶片倾角（Leaf Angle Distribution）对于NDVI也有干扰
    - 太阳高度角对NDVI也有影响，大气状况也会产生干扰
    - 随着植被覆盖度增加，NDVI不会线性增加，会发生饱和的现象

#### 太阳阻抗植被指数 ARVI
+ 太阳阻抗植被指数（Atmospherically Resistant Vegetation Index-ARVI）
+ 解决的问题：气溶胶可以同时增加红光和近红外的反射率，使NDVI偏低
+ 解决的方法：使用蓝光信息对红光和近红外波段数据进行修正
+ 遗留问题：经验系数如何取？需要动态调节

#### 垂直植被指数 PVI
+ 解决的问题：土壤对于植被指数的干扰
+ 解决的方法：找到垂直于土壤的分量

#### 土壤调节植被指数 SAVI
+ 解决的问题：土壤亮度对植被指数的干扰
+ 解决方法：引入土壤调节系数

#### 植被指数 MODIS EVI
+ 目前应用的最广泛的版本
+ 特性：	
    - 对土壤影响不敏感
    - 对亮度影响不敏感
    - 对气溶胶影响不敏感
    - 在高植被指数的不易饱和
    - 对叶面积指数、冠层类型、植被冠层结构更敏感
    - 对云、雪更敏感

#### 双波段 EVI
+ 一个妥协的产物
    - 对于土壤亮度干扰、饱和度问题比NDVI好一点
    - 受到气溶胶和云的干扰比EVI还大

### 线性变换在光谱分析中的应用
#### 穗帽变换 K-T变换
+ 别名：缨帽变换，K-T变换 Kauth-Thomas Transformation
+ 发现：植物生长过程在NIR-R图上运动轨迹为穗帽状

<img src="https://cdn.nlark.com/yuque/0/2024/png/2371504/1732080145648-dc5f0862-67cc-4236-8c3e-45dde8edfec4.png" width="622" title="" crop="0,0,1,1" id="ub360b086" class="ne-image">

+ 穗帽变换的三个坐标
    - 亮度：反映了总的辐射能 ，是所有波段的加权平均。
    - 绿度：反映了近红外与可⻅光之间的对⽐。
    - 湿度：主要反映⼟壤和植被中的⽔，其信息主要由短波红外与可⻅光近红外之间的差异提供。

<img src="https://cdn.nlark.com/yuque/0/2024/png/2371504/1732080332563-ef366898-a50f-47c8-abfd-9e79a290967b.png" width="541" title="" crop="0,0,1,1" id="ua8403242" class="ne-image">

+ 穗帽变换的本质是一种线性变换
    - 从多个光学维度，映射到亮度、绿度和湿度的三个维度
    - 通过加减来将对应的性质凸显出来

<img src="https://cdn.nlark.com/yuque/0/2024/png/2371504/1732080515947-033707c8-d085-4528-88c1-d646a592edc9.png" width="533" title="B1蓝，B2绿，B3红色，B4近红外，B5远红外1，B6远红外2" crop="0,0,1,1" id="ua5a6f7d3" class="ne-image">

+ 遗留问题：亮度、绿度和湿度两两之间还有相关性

#### K-L变换
+ 设计目的：
    - 用少数维度对光谱特点进行描述，实现数据压缩
    - 这些少数维度称为主成分
    - 找到新的一组正交基，使变异的在这一组基上的变异性体现在前几个维度上
    - 主成分对应的基需要两两正交
+ K-L变换A是一种特殊的正交变换，它满足：
    - 可以将输入信号X转化为输出信号Y
    - 输出信号Y的的协方差矩阵S_Y为对角矩阵，即输出信号Y的基是两两正交的。
+ 在[线性代数](https://baike.baidu.com/item/%E7%BA%BF%E6%80%A7%E4%BB%A3%E6%95%B0/800?fromModule=lemma_inlink)中，正交变换是[线性变换](https://baike.baidu.com/item/%E7%BA%BF%E6%80%A7%E5%8F%98%E6%8D%A2/5904192?fromModule=lemma_inlink)的一种，它从实[内积空间](https://baike.baidu.com/item/%E5%86%85%E7%A7%AF%E7%A9%BA%E9%97%B4/0?fromModule=lemma_inlink)V映射到V自身，且保证变换前后内积不变

### 其他植被指数
#### 光化学反射率指数 PRI
+ 反映了叶黄素循环的强度
+ 叶黄素循环：
    - 在强烈光照条件下，叶绿素可能吸收过多的光能，造成氧化性伤害，这称为光氧化损伤
    - 为了减轻这种损伤，植物通过调整叶绿体中叶黄素（xanthophyll）的形式来实现叶黄素循环
    - 额外的能量以热能释放。
+ 可见，叶黄素循环可以**表现植被光能利用效率**

#### 红边植被指数
+ 反映**植被被胁迫的程度**
+ 红边
    - 叶绿素对红光的吸收利用和叶片特殊结构共同导致了在700-750范围内反射率的陡然上升，称为红边（red edge）
+ 红边与胁迫的关系
    - 植被覆盖度越高，叶⾯积指数越⼤，红边斜率也就越⼤
    - 相应的植被生长状态越好，红边位置会出现“红移”；反之，则红边位置会相应“蓝移”
+ 观测红边的要求：高光谱分辨率，区分斜率

### 植被光学厚度：基于微波波段的指数
#### 理论基础：
    - 植被对电磁波的削弱主要是因为植物体内的水对电磁波的的吸收

<img src="https://cdn.nlark.com/yuque/0/2024/png/2371504/1732684780112-a924e3fd-28c7-4f39-8bd1-c63301077eea.png" width="359" title="" crop="0,0,1,1" id="ue0dd07b4" class="ne-image">

    - 亮温主要有植被和土壤贡献

<img src="https://cdn.nlark.com/yuque/0/2024/png/2371504/1732684090363-fa5fdeae-e2bf-4752-a651-cf22e34e2770.png" width="357" title="" crop="0,0,1,1" id="u66805527" class="ne-image">

+ 推导：
    - 热平衡时：发射率=吸收率
    - 透明物体：吸收率+反射率+透射率=1
    - 非透明物体：透射率=0
    - 综上热平衡非透明物体，如土壤：反射率 + 发射率 = 1
+ 反演方法：在某一时刻下空气温度代表植被和土壤亮温

<img src="https://cdn.nlark.com/yuque/0/2024/png/2371504/1732085342029-713238c7-0c72-46c0-9b4c-667af785667c.png" width="360" title="" crop="0,0,1,1" id="u5d705d64" class="ne-image">

#### 求解目的：获得植被光学厚度和土壤含水量
+ 植被光学厚度和植被透射率
+ 土壤发射率和土壤湿度有关
+ ⼟壤湿度，植被光学厚度的反演就是找到⼀组参数，使得正向模拟的亮度温度与观测到的亮温⼀致

#### 解决问题的方法：如何先解出一个未知数？
+ 双通道算法：H，V（极化）双通道，LPRM
+ 时间序列方法：考虑到 VOD 在较短时间内不会显著变化，但是⼟壤的介电常数 在短时间内变化可能变化较⼤k

### 叶绿素荧光
+ 荧光是光反应阶段的副产物
+ 荧光大约只占植物吸收太阳能 的2%左右，其峰值在690和740nm
+ 当植物结构完整时，大部分能量走光合作用通路，只有小部分走荧光通路。
    - 如果只有叶绿素，那么绝大部分都走荧光（20%）

利用氧气吸收线附近的变化，对叶绿素荧光进行检测。

这种方法只能用于近地面，因为如果经过一次大气又会被吸收。

如果想在卫星上进行观测，需要利用在大气吸收线以外的太阳夫琅和费暗线。
