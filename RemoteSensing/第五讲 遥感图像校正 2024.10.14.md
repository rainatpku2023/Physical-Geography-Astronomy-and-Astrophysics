### 图像基础
+ 多光谱信息：可以通过多个波段的信号强度的分布，推断被观测物体的性质
+ 遥感图像灰度分布统计
    - 概率密度分布
    - 累计直方图
    - 均值、方差、中值、偏度

### 几何纠正
+ 几何畸变：传感器生成的图像像元相对于地面目标实际位置发生了挤压、拉伸、扭曲和偏移，称为几何畸变。通过算法消除几何畸变的过程称为几何纠正
    - 传感器内部因素：透镜、探测原件、采样速率、扫描镜
    - 遥感平台因素：平台的高度、速度、轨道偏移、姿态变化
    - 地球因素：地球自转、地形起伏、地球曲率
    - 大气折射和投影的选择
+ 几何纠正：消除几何畸变的过程
+ 几何纠涉及的姿态：滚动、俯仰、航偏；高度、速度
+ 几何纠正需要解决的问题
    - 全景畸变：观测角度的影响（眼睛侧面看东西变形）
    - 地表曲率：地表曲率让地表“倾斜”了
    - 地球自转的影响：卫星与地表，相对速度变化，扫描的情况变化
+ 几何纠正步骤
    - 选取地面控制点与坐标变换
        * 选取标准：明显清晰的标识、不随时间变化、在一张影响上均匀分布、有一定的数量保证
        * 一部分用于校正，少部分用于验证
        * 目标：从图像到图像的转化，从图像到地图的转化

### 辐射校正
+ 辐射校正：消除或改正成像过程中附加在传感器输出的能量中的各种噪声的过程
+ 主要误差来源及修正方面
    - 传感器校正/辐射定标：传感器本身的性能引起的辐射误差
    - 大气校正：大气的散射和吸收引起的辐射误差
    - 地形校正：地形和光照条件发生变化引起的辐射误差

<img src="https://cdn.nlark.com/yuque/0/2024/png/2371504/1729055986219-19f45ede-21ae-4dcb-959c-8253f37b16a9.png" width="285" title="" crop="0,0,1,1" id="u6f4aee0a" class="ne-image">

+ 辐射定标：指建立遥感传感器的数字量化输出值DN与所对应视场中辐射亮度值之间的定量关系
    - 两个维度
        * 相对辐射定标：为了校正探测元件的不均匀性，需要对原始亮度均⼀化处理
        * 绝对辐射定标：建⽴DN值与实际辐射值的数学关系，获取地表辐射量。
    - 三个方法
        * 发射前实验室定标：实验室中调节
        * 基于星载定标器上的星上定标：观测自带的光源/太阳
        * 发射后的场地定标：观测已知地物
    - 辐射定标后的结果：可以获取地物的表观现象，包含了/混杂了地物的信息和大气的信息
        * 表观反射率：大气顶层的反射率
        * -> 需要进行大气校正
+ 大气校正：为了消除大气中大气分子和气溶胶的**散射和吸收作用**，短波长影响较大
    - 是否需要大气校正？
        * 不需要：只用单一的影响进行分析，而且大气比较均一，分析就不需要大气校正了
        * 需要：从遥感影像中反演各种生物、物理指标，需要进行大气校正，否则会有干扰
        * 需要：对比不同时间，或者大气均一性不强，需要进行大气校正
    - 大气校正的作用：根据原始数据，借助大气辐射校正，得到真实的地表反射率
        * 表观反射率：大气顶层的反射率
        * 地表反射率：由辐亮度图像计算得到，计算模型如辐射传输模型（出去云、大气、临近地物的干扰）
    - 大气校正的两个维度：
        * 绝对大气校正：消除大气的所有影响
        * 相对大气校正：消除大气对不同地区和时间数据影响的差异，使不同地区和时间的数据之间有一定的可比性；不一定需要完全去除大气影响，消除差异即可

+ 地形校正：消除地形效应
    - 地形校正的原因：DN与地表反射率的对应关系是基于地面全部水平的假定做出的；地表起伏会改变卫星、地表、太阳的相对校对位置关系

### 数字图像增强
+ 数字图像增强的目的：改善其视觉效果或者突出图像中的某些特征，从而更容易进行后续的分析或视觉解释
+ 数字图像增强的内部：辐射增强、空间增强、图像融合
+ 辐射增强
    - 线性对比度增强：线性拉伸适合亮度值正态分布或近正态分布，图像的亮度比较集中，只有⼀个峰值的情况
        * 改进方法：频率增强（分段函数）、特定低亮度增强
    - 直方图均衡：将原始图像的灰度图从比较集中的某个灰度区间分布在整个灰度空间中，不让图像都差不多灰

+ 空间增强
    - 低通滤波：
        * 目的与含义：滤掉（空间）高频变异，滤掉高频噪声
        * 本质与计算：按照原始周围和本身的信息，求加权平均（有点像地理学第一定律）
        * 要求：尽量保证矩阵中所有元素之和为1，即归一化
        * 改进方法：高斯滤波

    - 高通滤波：
        * 目的与含义：保留高频、抑制低频、让某一频率以上的信号分量通过，实现图像锐化 -> 找出差异，删去背景平滑
        * 变种1：横向边缘检测、倾斜45度边缘检测 -> 找出边缘
        * 变种2：Sobel算子求梯度

### 图像融合
+ 定义：将2张或2张以上的图像信息融合到1张图像上，使得融合的图像含有更多的信息，方便⼈来观察或者计算机处理。 -> 类似于GIS中的地理配准和计算
+ 图像融合⼀般包括像素级，特征级，和决策级融合
