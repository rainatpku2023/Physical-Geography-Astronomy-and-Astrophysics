### 目视解译（Visual Interpretation）
+ 概念：专业人员通过直接观察或借助相关辅助判读仪器在遥感图像上获取特定目标地物信息的过程，称为目视解译。
+ 直接解译标志：时间特征、空间特征、光谱特征
    - 具体来说有：亮度、颜色、大小/形状、纹理、空间上下文、阴影、空间格局、观测时间

<img src="https://cdn.nlark.com/yuque/0/2024/png/2371504/1730869378382-7ebf3586-59ed-461a-a31c-f17732eb7366.png" width="457" title="" crop="0,0,1,1" id="udf7ffe29" class="ne-image">

    - 颜色与亮度：通过图像的颜色与亮度，获取目标波段的反射率，获取信息
    - 大小和形状：根据尺度大小、几何形状获取其内容
    - 纹理：光滑还是粗糙？了解物体尺寸和像素分辨率的大小关系
    - 空间的上下文：根据目标区域与周围地物的关系，推测其性质
    - 阴影：根据太阳高度角、阴影长度计算出来目标地物的高度
        * 对于空间分辨率要求较高
    - 空间格局：分布规律
    - 观测时间：随着观测时间的变化，地表物体可能发生变化，在使用时需要注意年代
+ 间接解译标志：水系、地貌、土质类、植被、气候、人类活动
+ 遥感目视解译的具体方法：直接解译法、对比分析法、信息复合法、参数分析法、地理相关分析法

### 自动分类
#### 监督分类（Supervised Classification）
+ 监督分类的概念：根据一致训练区提供的样本，通过选择特征参数，建立判别函数，把图像中各个像元店规划到给定类别中的处理方法
+ 监督分类的主要问题
    - 分类问题：根据某些维度上的信息进行分类
    - 回归问题：根据观测变量得到目标变量的方法，e.g.反演植被的叶面积指数
+ 监督分类的主要步骤
    1. 选择训练样本
    2. 训练分类模型
    3. 应用分类模型分类
    4. 精度评价
+ 选择训练样本
    - 生成训练样本的两种方法
        * 目视解译+数字化训练样区
        * 选择随机点，电脑自动生成相似的地区
    - 注意事项：尽量不要混合不同类别地物，不希望分类时也不要用没有代表性的内容
+ 训练分类模型 -> 确定如何分类
    - 七种方法：平行六面体法、最小距离法、最大似然法、支持向量机、神经网络、决策树/随机森林、其他机器学习方法

#### 非监督分类（Unsupervised Classification）
+ 非监督分类的概念：根据图像数据本身的统计特征及点群的分布情况，不施加任何先验知识，从纯统计学的角度对图像数据进行类别划分的处理称为非监督分类。
    - 只是数学上的分类，所以又称为聚类分析
    - 聚类分析的优化：组间差异尽可能大，组内波动尽可能小
+ 非监督分类特点：非监督分类对不同类别进行划分，没有确定类别的属性，其属性时时候对个类别的光谱特性来确定，或与实地调查比较之后确定。
+ 非监督类型的困难：不知道分几类，这个需要提前知道
    - 解决方式：由聚类准则判断初始分类是否合理。不合理就修改分类，如此反复迭代运算

#### 混合方法（见ppt）
+ 通过结合不同的分类技术和方法，以提高分类的准确性和鲁棒性（Robustness）。
    - 结合：技术上的结合、波段上的混合、分类器上的混合、时间序列上的混合

#### 面向对象的机器学习
+ 卷积神经网络可以通过卷积运算，获取在一定空间范围内的组织信息（即纹理信息）

+ 地面真实（Ground Truth）：进行分类精度的方法

+ 精度评价指标
    - 总体精度（Overall Accuracy）：对角线上所有的样本像元数（正确分类的像元数）站所有像元的比例

### 变化检测
+ 变化检测：波段合成、代数运算、回归分析、主成分分析、分类后比较、变化向量分析、时间序列分析
+ 变化检测的五个维度：
    - 位置：Location
    - 时间：Time
    - 速度：Metric
    - 内容：Agent
    - 方向：Target
+ 变化检测的注意事项
    - 在不同波段的变化不同
    - 在不同空间分辨率的变化不同：例如如果分辨率太粗，小范围的变化可能看不出来
    - 在不同时间分辨率的变化不同：例如如果观测时间太久，可能无法注意到短期的洪水等等
+ 代数运算
+ 回归分析：本质上是归因分析，
    - 相⽐于直接做差，回归方法可以部分消除不同观测时间带来的系统偏差（e.g.，物候差异）
+ 主成分分析
+ 变化向量分析：不同波段组成一个空间，新坐标和旧坐标之间的连线，单位向量（方向）标志了变化的方向（Direction）；而模长标志了变化的强度（Intensity）
+ 时间序列分析：在像元尺度上进行分析，但是依赖于长时间的观测
    - 可以将不同时间尺度的波动分解进行分析
