### 地面平台
+ 高度一般小于300米，而且固定
+ 举例：高塔/车/船

### 航空平台
+ 高度一般在100-30000米之间
+ 举例：飞机/飞艇/无人机
+ 特点
    - 居高临下地观察
    - 可以记录动态影像
    - 扩大了光谱的感应范围
    - 可以提高空间分辨率和几何保真
+ 用途
    - 对太空飞行器上装载的遥感器进行模拟实验
    - 太空遥感的必要辅助手段
    - 直接进行各种遥感作业

### 航天平台
+ 在超出大气的地球附近或太阳系行星间飞行的飞行器，飞行高度超过卡门线
+ 特点
    - 更大的覆盖买你
    - 重访观测能力
    - 地面特征的定量测量
    - 半自动处理和分析能力
    - 成本相对较低，效益高

### 卫星轨道
+ 轨道分类：低地球轨道（1000-2000千米），中地球轨道（2000-36000千米），高地球轨道（36000千米以上）
    - 中地球轨道和高地球轨道的分界是地球同步轨道卫星
+ 开普勒第三定律：行星轨道半长轴的三次方和行星轨道周期的二次方成正比
+ 相关概念
    - 升交点：卫星由南向北运行时与赤道面的交点
    - 降交点：卫星由北向南运行时与赤道面的交点
    - 春分点：黄道面与赤道面在天球上的交点
    - 近地点：卫星距离地球最近的点
    - 远地点：卫星距离地球最远的点
+ 轨道要素
    - **轨道半长轴**
    - **轨道偏心率**
    - **轨道倾角：轨道平面与地球赤道平面的夹角（右手定则判断方向），分出来正方向卫星/反方向卫星/极轨卫星/赤道卫星**
    - **升交点赤经：升交点与春分点的夹角 防止轨道绕着地轴转动**
    - **近地点角距：半长轴在哪里？防止轨道绕着垂直于轨道面且穿过地心的轴转动。**
    - **真近地点角：卫星的位置**
+ 其他轨道参数：轨道高度、轨道周期、覆盖周期、星下点、星下点轨迹、同一天内相邻轨道在赤道的间距、每天绕地球的圈数
+ 对地观测轨道
    - 对全表面进行周期性观测成像覆盖：周期性的好处，同一地点不同日期被观测地点/太阳/观测卫星三者位置基本不变
+ 低地球轨道
    - 圆形轨道
    - 可以分为极轨和非极轨
        * 极轨卫星可以全球覆盖，每天到每个月一次的观测频率 -> 低纬度南方问到
        * 非极轨不能提供全球观测，观测频率从几小时到几周
    - 太阳同步轨道：在地球表面上同一个点上方时都在相同的平太阳时，即表面每次都接近相同的照明角度
        * 平太阳时：平太阳日直接除以24的结果，平太阳日时时假设太阳与地球距离不变
        * 轨道倾角大于90度：太阳同步轨道应该每天转0.9度，轨道倾角为98度时因为地球赤道处突起引力较大，可以让太阳同步轨道转起来
        * 轨道高度较低，发射难度小
            + 距离地表600-800km，周期约90-100分钟
        * 特殊：子午轨道与晨昏轨道 -> 一直在子午线上或者一直在晨昏线上；所以实际上太阳同步轨道周期与地球自转周期没有任何关系
    - 地球自转速度：每天东进360度；太阳光：每天东进1度左右

    - 增大重访周期的方法：同一个轨道上多放几个卫星
+ 中地球轨道
    - 中圆地球轨道：与地球静止轨道GEO和低轨道LEO相比，是一个成功的折中方案
        * 角速度较慢，接收信号时间更长，信号更稳定
        * 内范艾伦带以外
        * 可以用于定位
    - 定位系统：用四颗卫星求解自己的空间坐标（三个）和时间坐标（一个）
    - Molniya轨道
        * 非常大的偏心率，轨道倾角很大，大部分时间都游荡在北半球高纬度上空，而且总时间也显著增场
        * 这是一个俄罗斯的通信卫星
+ 高地球轨道
    - 地球静止轨道：至少需要三个才能完整观测赤道
    - 地球同步轨道：公转周期与地球自转周期相同的卫星。地球静止轨道是一种特殊的地球同步轨道
    - 倾斜地球同步轨道（Inclined Geosynchronous orbit）：在地球上看是在画8字，这样就稳定在某片地区上方了
    - 拉格朗日点
+ 卫星发射的考虑因素
    - 省油：纬度低一些
