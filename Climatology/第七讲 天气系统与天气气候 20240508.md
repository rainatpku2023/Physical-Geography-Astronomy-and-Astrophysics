### 1 气团及其天气气候
+ 气团：气象要素水平方向上物理性质均匀，垂直分布相似的空气团
    - 意思是水平上要基本一样，竖直上可以有差异但是这种差异不在水平方向上变化
+ 气团的形成条件：
    - 范围广阔、地表性质比较均匀的下垫面
    - 有利于空气停滞、缓行的流场
+ 气团的变性
+ 气团分类及特性
    - 地理分类（纬度地区）与热力分类（大陆还是海洋）
    - 大陆性的气团从大陆中产生，所以一般水汽含量比较少
    - 一般来说越靠近赤道越不稳定

    - 春秋：极地气团、热带太平洋气团

### 2 锋及天气气候
+ 锋：Front，气候交界线
+ 按照尺度进行分类
    - 气候锋
        * 行星锋：对流层中上层，环绕半球
        * 北半球分为极锋（冰洋气团和极地气团）和副热带锋（极地气团和热带气团）
    - 天气锋：地面及以上200-400千米，水平宽度几十千米

+ 按照气团的相对位置对锋进行分类
    - 冷锋：锋后冷气团起主导作用，推动着锋面向暖气团方向移动
        * 一型冷锋（缓行冷锋）：锋面移动较慢，坡度小，暖气团上升较慢
        * 二型风冷（急行冷锋）：封面移动较快，坡度大，暖气团上升较快
    - 暖锋：锋后暖气团起主导作用，推动着锋面向冷气团方向运动
    - 静止锋
    - 锢囚锋：有三个气团，分为暖式锢囚、中性锢囚和冷式锢囚

### 3 气旋（中心为低气压）及其天气气候
+ 气旋的大小和强度：中心气压，最外层闭合等压线
+ 气旋的辐合辐散结构：高空与低空配合，**<font style="color:#DF2A3F;">低空辐合高空辐散</font>**
+ 气旋的类型分类
    - 产生的地理位置：热带气旋、温带气旋 -> 类比台风
    - 结构特征：锋面气旋、无锋面气旋

<img src="https://cdn.nlark.com/yuque/0/2024/png/2371504/1715154582462-be6afa0c-d451-438a-8b1f-c3fea0e06ad7.png" width="347" title="冷锋画为蓝色三角形，暖锋化为红色半圆形，既有红色又有蓝色就是锢囚锋" crop="0,0,1,1" id="u8d123702" class="ne-image">

    - 温带锋面气旋：中心气压为970-1010hPa，范围1000千米左右
    - 热带气旋：范围600-1000千米左右，**<font style="color:#DF2A3F;">近地面最大平均风速</font>**和**<font style="color:#DF2A3F;">中心海平面最低气压值</font>**
        * 等级名称：热带低压、热带风暴、强热带风暴、台风、强台风、超强台风

<img src="https://cdn.nlark.com/yuque/0/2024/png/2371504/1715156360022-dc137c1a-c3d2-4868-9125-e7290ed087a2.png" width="628" title="" crop="0,0,1,1" id="ub66279f5" class="ne-image">

+ 温带锋面气旋的生成和源地：温湿气团和极地的干冷气团，在温湿性质不同的边界上生成
    - 气旋的源地分布基本上与纬圈平行，呈东西向
    - 在巨大山地的背风坡一侧及其以东地区
    - 海湾及内陆湖泊，冬季温度较高，很容易有气旋生成
+ 中国的锋面气旋
    - 北方气旋：蒙古气旋、东北气旋、黄河气旋
    - 南方气旋：江淮气旋、东海气旋
+ 气旋演化历程：初生阶段，成熟阶段，锢囚阶段，消亡阶段（温度槽跟上气压槽的过程）

+ 热带气旋的形成：第二类条件性不稳定，四个情况如果同时出现会发生正反馈调节
    - 高温洋面
    - 合适的纬度
    - 风的垂直切变小
    - 合适的流场：ITCZ，东风波，高层的辐散（高空抽吸，使现象可持续）

+ 热带气旋的结构：眼区、降水区、大风区、外围区
    - 书上的描述：外圈（大风区），中圈（涡旋区），内圈（台风眼区）
    - 天气：外螺旋云带，内螺旋云带，云墙，眼区

+ 热带气旋与其他天气系统的相互作用
    - 热带气旋会带来热带海洋上空的高温和高湿的空气
    - 比如冷锋与热带气旋相遇，会产生大量降雨
+ 影响中国的热带气旋：西太平洋热带气旋、南海
    - 生成地：菲律宾的东侧洋面上、关岛附近海面、南海中部

    - 大气环流场
    - 副热带高压（高压阻挡、低压吸引）
    - 科里奥利力的南北差异，因为热带气旋的尺度已经能达到了几个赤道度
    - 地形：折向、加速、跳跃

### 4 反气旋（中心为高气压）及其天气气候
+ 大小、强度衡量：中心气压、最外围等压线
+ 类型：温带反气旋、副热带反气旋、热带反气旋；冷性反气旋、暖性反气旋
    - 温带+冷性反气旋--寒潮天气
+ 中国的冷高压：
    - 西北向：从西伯利亚西部和蒙古国进入我国，冬季
    - 开始从北向南或者从东北向西南移动，然后在北纬55度转向东南移动，有的经过西伯利亚西部和蒙古国进入我国华北地区或者从西伯利亚进入我国东北地区，冬季
    - 内蒙古生成冷性反气旋，常直接南下进入我国，冬季
    - 北纬50度以南，亚洲大陆的西方移来，多由西向东移动直接侵入新疆地区，还有的则再折向东经内蒙古进入我国，夏季
+ 伴随的天气：寒潮
    - 寒潮的源地和路径
        * 新地岛以西的北冰洋洋面 40%
        * 新地岛以东的北冰洋洋面 18%
        * 冰岛以南的大西洋洋面 33%
    - 寒潮过境的环流特征：
        * 高空：500hPa，一个低压槽（地面冷高压为浅薄系统），后部为等温线密集区；850hPa高空锋区
        * 地面天气形势，寒潮冷锋，后面为强大冷高压
    - 超级寒潮产生原因：北极漩涡南下

+ 中国大型降水系统

### 5 中、小尺度的天气系统
+ 特点：空间尺度小、生命时间短、气象要素场梯度值大、天气现象激烈
+ 雷暴
    - 一般雷暴：只伴有阵雨
    - 强雷暴：伴有暴雨、大风、冰雹、龙卷等严重灾害性天气现象之一
    - 一次雷暴过程并不是只有一块雷暴云，而往往是由几个或者更多个处于不同发展阶段的雷暴单体所组成
    - 内陆多于沿海，山地多于平原，夏季多于冬季
+ 飑线、龙卷、冰雹
    - 区分水龙卷和尘龙卷，后者没有降水
    - 龙卷中心附近有下沉气流，自中心向外是强盛的上升气流，组成漏斗状云体
    - 雷暴时间感觉有凉风
    - 对流性天气系统：雷暴、飑线、龙卷

<img src="https://cdn.nlark.com/yuque/0/2024/png/2371504/1716362858598-015e91a8-f31f-43b9-bd56-6aaa8a7c170f.png" width="529" title="" crop="0,0,1,1" id="ubacaf30e" class="ne-image">

    - 闪电的种类：云和云之间的闪电，云和地面的闪电，云内部风摩擦产生的闪电
    - 龙卷风过境时气压会明显下降
    - 龙卷风基本上发生在北美的原因：山地为南北走向，而且两侧都有海洋，龙卷风廊道使北冰洋冷气流和来自大西洋的暖湿气流可以顺利会和，雷暴和龙卷风
