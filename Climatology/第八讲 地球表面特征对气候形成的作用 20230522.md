### 1 海陆分布对气候的影响
+ 海陆表面的热力性质差异
    - 反射率
    - 吸收辐射层的厚度
    - 辐射平衡
+ 海陆表面的热量转换特性
    - 土壤及海洋中的热量转换：热量传输的差异、导温系数的差异、活动层广度差异
    - 海陆与大气之间的热量交换：海洋多、陆地少
    - 海路表面的蒸发耗热
+ 海洋对气候的调节作用
    - 海洋的储热作用
    - 海洋的水源作用
    - 洋流的气候的影响

+ 海陆影响形成气候要素的主要差异
+ 温度差异：平均气温、气温较差、气温相时、春温与秋温
+ 环流的差异：季风和海陆风（季风尺度更大，海陆风尺度较小）
+ 湿润情况的差异：降水差异、干湿气候

### 3 冰雪覆盖对气候的影响
+ 冰雪覆盖类型：季节性雪被、高原冰川、大陆冰盖、永冻土、海冰
+ 世界冰雪覆盖状况：地球总水分量的21.5%，淡水量的80%
+ 冰雪覆盖与气温
    - 高反射率
    - 黑体长波辐射
    - 冰雪表面与大气之间的热量交换能力很弱，特别是海冰对海洋显热和潜热的隔离效应
    - 冰雪表面的饱和水气压低于同温度的水面，冰雪对空气的致干作用
    - 融冰化雪消耗能量

### 4 人类活动对气候的影响
+ 大气成分改变对气候的影响
    - 人类活动影响大气中化学成分的变化
        * CO2和CH4逐年增加
        * 臭氧减少
        * 氯氟碳化物从无到相当量级
+ 下垫面性质与局地气候的形成
    - 地表植被变化对气候的影响
        * 地表反射率减小
        * 植物冠层蒸发能力强
        * 植被截留降水，减小地表，增加土壤蓄水
        * 降低风速
+ 人类活动形成的特殊气候
    - 防护林的气候效应
    - 城市气候
        * 浑浊岛：污染物多，凝结核多，低云多，雾多
        * 城市热岛

### 5 季风
+ 季风的特点
    - 盛行风向随季节变化 -> 一般以120度为分界线
    - 不同季节气团性质不同 -> 一般观察夏季和冬季之间的对比
    - 天气现象有本质性的季节差异 -> 气温、湿度、降水等
+ 衡量季风：环流指数和风速指数	
+ 季风的标准
    - 1月和7月盛行风向的变化至少有120度
    - 1月和7月盛行风向的平均频率超过40%
    - 1月和7月中至少一个月的平均合成风速风速为3m/s
        * 平均合成风速 -> 加权的矢量平均
+ 季风的成因
    - 海陆热力差异对季风形成的作用
    - 行星风带的季节移动 <- 天文辐射的季节性变化
    - 高原大地形的影响 -> 改变气压分布、改变气流运动方向
