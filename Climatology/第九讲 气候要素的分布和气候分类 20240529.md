### 1 气温和降水及气压系统的地理分布
+ 海平面气温的地理分布
    - 影响因素：维度因素、海陆和地形的影响
    - 描述方式：等温线走向、热赤道、极值温度的分布
+ 降水的地理分布
    - 赤道多雨带：哈德莱环流的上升区，对流雨为主，1000-2000mm
    - 南北纬15-35度少雨带：副热带高压
    - 中纬度（温带）多雨带：锋面、气旋雨，500-1000mm
    - 高纬少雨带：<300mm

### 2 气候分类的基本原理
+ 基本原则：以纬度地带性规律为背景，非地带性规律打破
    - 纬度地带性规律：天文气候带、温度带、环流三风四带
    - 非地带性规律：海陆差异和大地形，海拔高度
        * e.g.冷洋流和暖洋流对层结稳定性影响不同
        * e.g.珠峰南北坡气候带变化情况，南坡各个海拔高度会有不同的气候状况

### 3 世界气候分类方法
+ 实验分类法-柯本分类法
    - 依据：以气温、降水及其季节变化为分类依据，参考植被景观 -> 例如在哪个温度植被景观发生明显变化就在哪里划分
    - 分类等级：气候带、气候型、气候副型
    - 不足：干燥气候、高山气候

+ 成因分类法-阿里索夫分类法
    - 分类依据：综合考虑太阳辐射、下垫面性质、海陆分布、洋流、大气环流和水分输送	
        * 例如，阿里索夫分类法用的是气团和气候锋
    - 分类系统：气候带（过渡带）、气候型

+ 理论分类法-斯查勒气候分类
    - 分类依据：在深入认识气候特征与气候型成因子的关系基础上，应用水、热平衡理论
    - 分类体系：利用了很多对准确性要求很高的观测数据
        * 气候带 <- 可能蒸散量
        * 气候型 <- 土壤缺水量
        * 气候副型 <- 土壤出水量和土壤多余水量
