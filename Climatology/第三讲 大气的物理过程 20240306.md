#### 辐射的基础知识
能量越短能量越大

辐射：以电磁波的形式传递能量的方式，对应的能量为辐射能

<font style="background-color:#FBDE28;">辐射通量：单位时间内，通过任意表面的辐射能</font>

辐射通量密度：单位面积的辐射通量

辐射强度：单位时间内，垂直于任何方向的单位面积，或者此方向上立体角内的辐射能，具有很强的方向性

普朗克定律：黑体辐射强度与波长、温度的关系。

维恩位移定律：辐射能力最大的波长随温度升高向短波移动。

斯蒂芬-玻耳兹曼定律：**黑体**总的通量密度与绝对温度的四次方成正比。

**<font style="color:#DF2A3F;">基尔霍夫定律：物体如果能够吸收某一波长的辐射就必然能强烈发出同一波长的辐射。</font>**

### 地球和大气的辐射
+ 地面辐射
+ 大气辐射
    - 两个方向：向上向太空中（大气辐射），向下向地面（大气逆辐射），二者均为长波辐射
    - 大气中吸收辐射的物质：主要为二氧化碳和水，臭氧能强吸收9-10微米
    - 8.5-12微米波段大气几乎不吸收，这个现象称为大气窗

+ 地球和大气向上射出的长波辐射
    - 与地表的性质有关，与大气的性质也有关
+ 地面有效辐射：地面辐射量-大气向下辐射量
    - 影响因子：大气情况（云等），大气温度和地面温度
    - 可分析年变化和日变化，季节变化
        * 干季地面辐射大，大气逆辐射小，所以地面有效辐射大；干季地面有效辐射日起伏大
        * 湿季地面辐射小，大气逆辐射大，所以地面有效辐射小；湿季地面有效辐射日起伏小

### 地面辐射平衡
+ 本质上是收入和支出的关系
+ 日变化规律
    - 日出后一段时间，日落前一段时间平衡为0，收支瞬时平衡
    - 最大值在中午出现，此时收入极大
    - 最小值在夜间出现，此时支出较大
+ 年变化规律
    - 夏季收入多，冬季支出多，净盈余，盈余部分通过热力学过程传给大气
+ 全球分布规律：与地表性质有关，纬度位置有关，与上方大气厚度有关，与上方大气组成有关（参考前几章）

### 大气辐射平衡
收入有吸收的太阳辐射，从地面吸收的长波辐射；支出有射出的长波辐射；净亏损，亏损部分通过热力学过程从地面获得补齐

#### 3.4 气层稳定度和气温的垂直变化
+ 气温的垂直变化：
    - 环境的气温变化：周围空气的温度直减率，符号γ，是一个大背景
    - 气团的气温变化：干绝热直减率γd，湿绝热直减率γm，一般来说γm<γd
+ 气层稳定度
    - 书上定义：大气稳定度是指气块受到任意方向的扰动后，返回或远离平衡位置的趋势和程度
    - 个人理解：当有微小扰动时，位置是正反馈还是负反馈
+ 两种变化
    - 绝热变化：只有做功，没有热传递
        * 当气团运动速度很大时，气块来不及被加热或者冷却，认为是绝热变化
        * 用温度表征气体环境的合理性：若温度高，一般来说水汽多，那么密度大，密度大的结果就是向下回去
        * 类型：干绝热过程（没有相变），湿绝热过程（有相变）
    - 非绝热变化：有热传递
+ 大气的层结：看温度、湿度和高度的分布
+ 五种情况
    - 绝对不稳定：大气环境垂直减温率大于干空气
    - 绝对稳定：大气环境垂直减温率小于湿空气
    - 条件性不稳定
+ 总结：大气空气气温直减率γ越大，大气层的结构越不稳定
+ 近地层逆温：
    - 辐射逆温：在晴朗无云或少云的夜间，地面很快辐射冷却，贴近地面的气层也就随之降温

    - 地形逆温：
    - 平流逆温：暖空气平流到冷的地面或者水面上，发生接触冷却作用，越接近地面降温越多，因此形成逆温现象。平流的风速较大时，会引起湍流逆温，进一步加强逆温效应。

    - 湍流逆温：低层空气与湍流混合而形成的逆温，湍流运动中上升空气的温度是按照干绝热直减率变化（ 比气温直减率大），使得上层空气降温下层空气升温。因此在混合层上方形成了逆温层。
    - 下沉逆温：气层的相对位置不发生变化，但是因为下沉过程气压增大、气层辐散，顶部气层下降远，顶部气层升温多于底部，产生逆温。
    - 锋面逆温：冷暖气团相遇时，较轻的暖空气爬到了冷空气上方，在界面附近产生逆温层。
