### 第一节 气候系统
#### 1 气候概念的发展
古典气候：大气要素的长期平均（30年）

现代气候：气候的形成/变化与预测

以气候系统为研究对象：大气/海洋/冰雪/大陆/生物圈



+ 1975年，第一次用系统的观点看气候问题，作为独立的学科，不作为二级学科下设
+ 1975-1996-2005年，气候系统描述更加复杂
+ 2020作为地球系统进行研究，增加人类的活动研究（e.g.可持续发展问题）



气候学：主要研究全球和区域气候季节内/季节间/年际以及更长的代际和更长的时间变化规律和成因/全球气候系统各圈层相互作用过程和<font style="color:rgb(0, 0, 0);">以及</font><font style="color:rgb(192, 0, 0);">人类活动</font><font style="color:rgb(0, 0, 0);">对气候的影响。它的研究对象具有</font><font style="color:rgb(192, 0, 0);">综合性</font><font style="color:rgb(0, 0, 0);">。</font>



#### 2 气候系统的研究方法
+ 气候监测：收集当下的数据
    - 大气常规检测：自动记录
    - 其他系统的常规检测：如海洋等
    - 非常规检测：需要自己分析
+ 气候诊断：诊断异常，分析异常出现的原因，为建模打基础
+ 气候重建：向之前时间延伸，研究之前的气候
    - 树木年轮，冰芯，孢粉，珊瑚，史料分析，其他
    - 史料分析：可能要分析很长时间的数据，还要收集许多地方史料（不仅包含以记录为目的的记录，还有许多其他目的的纪录，如赈灾请求等）。
+ 气候模拟：数学建模
    - 分块模拟，分的越细模拟越准，但是对计算资源消耗更大

### 第二节 主要气象要素及观测/分析
气象要素：表示大气状态的物理量和物理现象，其中物理量需要仪器观测，而物理现象采用目测

两种观测手段得到的两组数据同时使用时需要用到均一化分析，例如不同时代的风向标得到的数据能不能同时使用。

#### 1 气象要素及观测仪器
+ 气温：表示大气冷热程度的物理量
    - 特殊仪器：最高温度温度计（带有缩口，类似于水银体温计）；最低温度温度计（不能使用水银，一般使用煤油+浮标的方法）
    - 平均温度的产生方式：最高温与最低温的平均？每日4次测量，每日8次测量，每日24次测量还是每日48次测量的平均？
    - 城市热岛：钢筋水泥的热岛效应与强排放的热岛效应
+ 湿度：大气湿度即大气中的含水量，当然湿度还有土壤湿度
    - 衡量方式：水汽压/饱和水汽压/饱和差/绝对湿度/相对湿度/比湿/混合比

+ 气压
+ 三个量的统计分析：时值，日值，月值，年值；最值分析，极值分析，变化过程分析，平均值
    - 问题：平均值的产生方式如何，两组数据之间是否相同？数据的收集标准是否相同？（如仪器的放置位置等
    - 气象收据好获得，但是可能有潜在的坑
+ 风：空气的运动，三维矢量
    - 风速大小用米每秒衡量，也可以用风的级数表示，级数和米每秒并非线性关系
    - 风速的方位有360度方向，16方位，8方位等等，可绘制成为风向玫瑰图。不同月份的玫瑰图可以对比研究。
+ 降水：空气中降落的水量，单位为毫米
    - 最终都是用融化出来的水计算
    - 仪器：最简单的就是量筒，目前进化到了称重式的仪器，可能换算的并不是非常准？
    - 数据记录的"0"值问题：实际上就是0，没有测出来，有数但是因为量太小测不出来，数据缺失

### 第三节 大气成分和垂直分层
#### 大气成分
+ 常定成分：氮气/氧气/惰性气体
+ 可变成分：二氧化碳/一氧化碳/甲烷/臭氧/二氧化硫/一氧化二氮/二氧化氮等
    - 气溶胶：自然的火山尘埃/宇宙尘埃/海水溅沫/花粉/种子/地面尘埃/沙尘暴等；人为的粉尘和水汽

边界层：对流层的最底部，会和地表进行相互作用。边界层之上是自由大气。

温度分布规律：对流层随高度上升而下降，平流层随高度上升而上升，中间层随高度上升而下降，热层温度回升。
